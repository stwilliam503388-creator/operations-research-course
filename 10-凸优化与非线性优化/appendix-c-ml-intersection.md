# 附录 C：凸优化与机器学习

机器学习训练本质上经常是优化问题：

```text
minimize  损失函数 + 正则化项
```

## 常见连接

| 机器学习模型 | 优化视角 |
|--------------|----------|
| 线性回归 | 最小二乘凸二次问题 |
| Ridge 回归 | 加 L2 正则的凸优化 |
| 逻辑回归 | 凸 log loss + 正则化 |
| SVM | 凸二次规划或铰链损失优化 |
| 神经网络 | 大规模非凸优化 |

## 为什么这重要

- 凸模型更容易诊断和复现
- 非凸模型表达能力强，但更依赖初始化、数据和训练技巧
- 正则化可以看成对参数复杂度的惩罚
- 优化指标下降不等于业务指标变好

## 如果你只记住一件事

机器学习里的“训练模型”，很多时候就是“解一个优化问题”；理解优化，能帮你判断 loss 曲线、正则化和收敛异常。
