# 08 综合项目：营销预算连续优化

## 项目目标

你要为三个渠道分配营销预算：搜索广告、社交广告、内容投放。每个渠道都有边际收益递减，预算总额有限，还要避免过度集中。

你的任务是建立一个连续优化模型，比较至少两种算法，并做敏感性分析。

## 建模要求

至少包含：

- 决策变量：三个渠道预算
- 目标函数：最大化总转化收益减去投放成本
- 约束：总预算、单渠道上下限、非负
- 凸性或非凸性判断
- 至少两种算法对比
- 参数敏感性分析
- 现实边界说明

## 推荐模型

使用凹收益函数：

```text
revenue_i(x_i) = a_i log(1 + b_i x_i)
```

最大化凹函数等价于最小化凸函数的相反数：

```text
minimize -Σ_i revenue_i(x_i)
subject to Σ_i x_i ≤ budget
           lower_i ≤ x_i ≤ upper_i
```

## 代码骨架

完整代码在 `code/capstone.py`，包含：

- `generate_channels()`
- `objective()`
- `solve_with_slsqp()`
- `solve_with_projected_gradient()`
- `sensitivity_analysis()`
- `main()` 中的断言式检查；整门课统一检查入口见 `code/run_checks.py`

## 验收标准

- 输出每个渠道的预算
- 检查总预算约束满足
- 检查目标值优于平均分配 baseline
- 比较 SLSQP 和投影梯度结果
- 做至少 3 个预算水平的敏感性分析
- 说明现实中还需要考虑归因误差、投放延迟和渠道相互影响

## 如果你只记住一件事

Capstone 的重点不是写一个复杂求解器，而是完整走通：业务问题 → 模型假设 → 连续优化 → 代码求解 → 结果解释 → 现实边界。
