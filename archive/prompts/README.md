# 课程生成提示词归档

这里保存从各课程学习目录中移出的提示词和生成过程文件。

**注意：这里是历史生成物归档。文件名、目录清单和实现计划可能已经过期，学习时一律以根目录 `README.md` 和各课程 `README.md` 为准。**

归档保留原始相对路径，例如 `08-随机规划与鲁棒优化/prompt.md` 会放在 `archive/prompts/08-随机规划与鲁棒优化/prompt.md`。
