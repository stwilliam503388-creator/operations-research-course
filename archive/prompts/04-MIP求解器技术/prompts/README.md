## 关键设计决策

1. **先直觉后公式**：每个抽象概念强制先给生活类比 → 白话解释 → 最后才出公式
2. **双类比策略**：关键概念必须配 1 个生活类比 + 1 个工作类比
3. **逃生通道**：每章末尾必须有「🆘 卡住了怎么办」，确保读者不会在中途放弃
4. **分文件输出**：v4 将单次对话生成改为 19 个独立 Markdown 文件生成，读者拿到的是一个完整可导航的课程目录
5. **自检锚点**：每个文件写完 LLM 必须执行「奶奶测试」+「术语密度测试」

## 使用方法

如果要复制这套流程生成其他主题的教程：
1. 用 `05-prompt-v4-file-output.md` 作为基础模板
2. 替换主题、案例和约束为你的目标领域
3. 用 `06-execution-plan.md` 的并行策略规划生成过程
4. 生成完后用 Phase 5 的全量验证步骤检查
