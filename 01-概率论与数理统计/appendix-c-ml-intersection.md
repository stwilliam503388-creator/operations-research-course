# 附录C：概率论与机器学习的交叉点

> 如果你是 ML 从业者，以下是概率论在 ML 中最关键的六个交叉点。

---

## 1. 最大似然估计 = 很多损失函数的源头

| ML 任务 | 概率假设 | 损失函数 |
|---------|---------|---------|
| 线性回归 | 误差 ~ N(0, σ²) | 均方误差 MSE |
| 逻辑回归 | 输出 ~ Bernoulli(p) | 交叉熵损失 |
| 分类（多类）| 输出 ~ Categorical | 多类交叉熵 |

**理解**：当你最小化 MSE 时，你实际上在做"假设误差是正态分布下的最大似然估计"。损失函数和概率模型是对应的。

## 2. 贝叶斯机器学习

频率学派 ML：给出一组最优参数 θ̂。
贝叶斯 ML：给出参数的后验分布 P(θ|D)。

**优势**：
- 不确定性估计：你不仅知道"预测值是 5"，还知道"95% 可信区间是 [3,7]"
- 自动复杂度控制：贝叶斯模型比较可以自动避免过拟合
- 小样本学习：先验知识提供额外信息

**代表方法**：高斯过程、贝叶斯神经网络、贝叶斯线性回归

## 3. 概率图模型

用图结构表示随机变量之间的依赖关系。

| 模型 | 图类型 | 典型应用 |
|------|-------|---------|
| 朴素贝叶斯 | 有向（特征→标签）| 文本分类 |
| 隐马尔可夫模型 | 有向链 | 序列标注、语音识别 |
| 条件随机场 | 无向 | NER、分词 |
| 贝叶斯网络 | 有向无环图 | 专家系统、因果推断 |

## 4. 蒙特卡洛方法在 ML 中

| 方法 | 用途 |
|------|------|
| MCMC（马尔可夫链蒙特卡洛）| 从难以直接采样的后验分布中采样 |
| 变分推断 | 用优化代替采样来近似后验 |
| 重要性采样 | 强化学习中的策略评估 |
| 重参数化技巧 | VAE 中允许反向传播通过随机采样 |

## 5. 置信区间 vs 预测区间

| | 置信区间 | 预测区间 |
|--|---------|---------|
| 对象 | 参数（如回归系数）| 新观测值 |
| 范围 | 较窄 | 较宽（包含随机误差）|
| 在 ML 中 | 特征重要性？| 预测的不确定性 |

## 6. 概率校准

很多 ML 模型输出的"概率"并不是真正的概率。

**例**：模型预测"这个邮件是垃圾邮件的概率是 0.9"。如果所有预测为 0.9 的邮件中实际有 90% 是垃圾邮件，模型就是校准好的。

**常用校准方法**：Platt Scaling、Isotonic Regression

---

## 一句话总结

概率论是 ML 的理论基础：损失函数来自似然，不确定性来自后验，采样来自蒙特卡洛，推断来自图模型。
