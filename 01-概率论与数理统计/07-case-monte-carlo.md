# 案例5：蒙特卡洛仿真与风险评估 ★★★★☆

> 评估一个投资组合在 95% 置信水平下的最大可能损失（VaR）。

---

## 场景

你是一个小型投资团队的分析师。你的投资组合包含两种资产：
- **股票 A**：年化收益率 12%，标准差 20%
- **股票 B**：年化收益率 8%，标准差 10%
- **两者相关系数**：0.3

配置比例：60% A，40% B。初始投资额 100 万元。

老板问："一年后最差会亏多少？"

## 建模

### 资产组合的期望与方差

```
E[Rp] = w₁μ₁ + w₂μ₂ = 0.6×0.12 + 0.4×0.08 = 10.4%

Var(Rp) = w₁²σ₁² + w₂²σ₂² + 2w₁w₂ρσ₁σ₂
        = 0.36×0.04 + 0.16×0.01 + 2×0.6×0.4×0.3×0.2×0.1
        = 0.0144 + 0.0016 + 0.00288
        = 0.01888

σ(Rp) = √0.01888 ≈ 13.74%
```

### VaR（Value at Risk，风险价值）

VaR 是"在给定的置信水平下，最大可能损失"：

```
95% VaR = μ - 1.645σ（正态分布假设下）
        = 10.4% - 1.645 × 13.74%
        = 10.4% - 22.6%
        = -12.2%
```

即：**95% 的概率下，你的最大损失不超过 12.2 万元。**

但这是假设收益率服从正态分布。如果真实分布有厚尾，VaR 会被低估。

### CVaR（Conditional VaR，条件风险价值）

CVaR 是"当损失超过 VaR 时，平均损失是多少"：

```
CVaR = E[损失 | 损失 > VaR]
```

CVaR 比 VaR 更保守——它考虑了尾部风险。

## 方法

**蒙特卡洛仿真步骤**：

1. **生成相关随机变量**：用 Cholesky 分解生成两组相关正态变量
2. **模拟收益率**：对每个资产，用 μᵢ + σᵢ × Zᵢ 模拟一年收益率
3. **计算组合收益率**：加权平均
4. **计算组合价值**：100万 × (1 + Rp)
5. **重复 N 次**：N = 10000, 50000, 100000
6. **分析结果**：
   - 计算 VaR（排序后取 5% 分位数）
   - 计算 CVaR（小于 VaR 的损失的平均值）
   - 绘制收益率分布直方图

1. ✅ 仿真的期望值接近解析解（大数定律）
2. ✅ 仿真次数增加时估计误差减小（误差 ∝ 1/√n）
3. ✅ 95% VaR ≈ μ - 1.645σ（正态分布时与解析解一致）
4. ✅ CVaR < VaR（CVaR 是更保守的风险度量）

## 洞察

1. **VaR 有缺陷**。它告诉你"95% 的情况下你不会亏超过 X"，但没有告诉你"那 5% 的情况下你平均亏多少"。这就是 CVaR 存在的意义。

   **金融危机中的教训**：2008 年之前，银行的 VaR 模型说"99% VaR = 很小"。但模型用的是正态分布——而真实市场有厚尾。那 1% 的尾部事件发生了，而且损失远大于 VaR 的预测。

2. **蒙特卡洛 vs 解析解**：
   - 解析解（公式）更快、更精确——但当模型复杂时无法用公式
   - 蒙特卡洛更灵活——可以处理任何分布、任何非线性关系
   - 实战中：先用解析解做快速估计，再用蒙特卡洛做精细验证

3. **收敛速度很慢**。要去掉一个零（误差从 1% 到 0.1%），需要 100 倍的仿真次数。这就是为什么需要方差缩减技术。

4. **方差缩减：对偶变量**。每生成一个标准正态变量 Z，同时使用 Z 和 -Z 来做两次仿真。这可以显著降低方差（对对称分布尤其有效）。

## 延伸

- 尝试非正态分布（t 分布、混合分布）— 看看 VaR 如何变化

- 实现更高级的方差缩减：重要抽样、控制变量
