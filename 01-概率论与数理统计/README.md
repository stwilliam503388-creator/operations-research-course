# 概率论与数理统计基础课程

> 一个周末，从零到能理解概率论的核心概念、看懂随机变量的分布、会用蒙特卡洛仿真评估不确定性——为运筹学、博弈论、随机规划打下数学基础。

---

## 内容质量标准

本课程遵循根目录的 [课程内容质量规范](../COURSE_CONTENT_GUIDE.md)：用人话解释核心直觉，同时明确模型假设、适用边界和可运行验证方式。

## 这份教程是什么

这是一份给**有高中数学基础、正在学习OR/博弈论/随机规划、但概率知识停留在"背公式"阶段**的技术人写的自学教程。它不讲学院派的测度论——每一章都围绕「这个概率概念在优化/决策中有什么用」展开。

读完以后，你一定会：
- 区分清楚 PDF、PMF、CDF，知道什么时候用哪个
- 看到"需求服从正态分布"就能算安全库存和服务水平
- 用蒙特卡洛仿真评估一个随机决策方案的风险
- 理解大数定律和中心极限定理为什么是统计学的灵魂

## 术语速查表

### 核心术语

| 中文 | English | 一句话人话 |
|------|---------|-----------|
| 样本空间 | Sample Space | 像把所有可能结果先摊在桌上：掷骰子就是 `{1,2,3,4,5,6}` |
| 概率 | Probability | 像给每个结果贴一个“会不会发生”的可能性标签，0 是绝不，1 是必然 |
| 条件概率 | Conditional Probability | 已经知道一件事发生后，再问另一件事有多可能；像“下雨了，堵车概率多大” |
| 随机变量 | Random Variable | 把随机事件翻译成数字，方便计算；比如“掷到几点”就是一个数字 |
| 概率质量函数 | PMF | 离散结果的概率清单：每个点分别有多大概率 |
| 概率密度函数 | PDF | 连续变量的“概率浓淡图”；某一点不是概率，一段面积才是概率 |
| 累积分布函数 | CDF | 像从左往右累计概率，告诉你“结果不超过 x”的机会有多大 |
| 期望 | Expectation | 如果重复很多很多次，最后平均下来大概是多少 |
| 方差 | Variance | 看结果有多不稳定：方差越大，像坐过山车，起伏越猛 |
| 正态分布 | Normal Distribution | 像中间高、两边低的钟形曲线，身高和测量误差常常长这样 |
| 大数定律 | Law of Large Numbers | 试得越多，平均值越不容易瞎晃，会慢慢靠近真实水平 |
| 中心极限定理 | Central Limit Theorem | 很多小随机因素加在一起，平均值常会变成钟形曲线 |
| 假设检验 | Hypothesis Testing | 像法庭审案：先假设“没效果”，再看证据够不够推翻它 |
| 置信区间 | Confidence Interval | 用样本画一个“真值可能在哪”的范围，但不是说真值会随机跳来跳去 |
| p值 | p-value | 如果“没效果”是真的，现在看到这么极端结果有多奇怪 |

### 进阶术语

| 中文 | English | 一句话人话 |
|------|---------|-----------|
| 贝叶斯定理 | Bayes' Theorem | 像侦探破案：看到新证据后，重新调整每个嫌疑人的可能性 |
| 协方差 | Covariance | 看两个东西是不是一起涨、一起跌，还是一个涨一个跌 |
| 相关系数 | Correlation | 把协方差压到 `[-1,1]` 的刻度尺上，方便比较关系强弱 |
| 矩母函数 | MGF | 像一个“统计量生成器”，期望、方差等信息可以从里面拆出来 |
| 充分统计量 | Sufficient Statistic | 像把原始数据压缩成一张小纸条，但关键参数信息还没丢 |
| 极大似然估计 | MLE | 反推“什么参数最可能生成我眼前这堆数据” |
| 贝叶斯估计 | Bayesian Estimation | 先带着经验判断出发，再用新数据修正这个判断 |
| 马尔可夫链 | Markov Chain | 像只看当前位置导航：下一步去哪，只看现在，不翻旧账 |

## 学习验收标准

学完本课程后，建议用下面 5 条自检：

- 能区分 PMF、PDF、CDF，并说明连续随机变量“某一点概率为 0”是什么意思。
- 能解释置信区间和 p 值的严格含义，不把它们说成“真值概率”或“原假设概率”。
- 能根据均值、方差和分布假设，判断库存、安全库存、风险评估中该算什么量。
- 能运行 Python 蒙特卡洛案例和 C++ 对照案例，并解释估计误差为什么随样本量下降。
- 能指出“需求服从正态分布”等教学假设在真实业务中什么时候会失效。
