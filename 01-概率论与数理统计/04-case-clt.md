# 案例2：中心极限定理演示 ★★☆☆☆

> 从一个非正态分布中重复抽样，展示样本均值的分布如何随样本量增加趋近正态。

---

## 场景

你是运筹学课程的助教。学生们不理解为什么 CLT 这么"神奇"——从一个偏态分布里抽样，均值竟然会是正态的。

你的任务：用一个简单的 Python 脚本，直观地展示 CLT 在不同分布、不同样本量下的收敛过程。

## 建模

### CLT 的核心公式

如果 X₁, X₂, ..., Xₙ 是独立同分布的随机变量（均值 μ，方差 σ²），则：

```
X̄ ~ N(μ, σ²/n)  近似，当 n 足够大时
```

标准化后：
```
Z = (X̄ - μ) / (σ/√n) ~ N(0, 1)
```

### CLT 的魔力

- 原始分布可以是**任何形状**：偏态、双峰、均匀、指数
- **n=1** 时：样本均值的分布 = 原始分布（还没开始收敛）
- **n=30** 时：经验法则——大多数分布下均值已经接近正态
- **n=100** 时：即使是很偏态的分布，均值也基本正态了

### 需要多少个样本？

这取决于原始分布的"奇怪程度"：

| 原始分布 | 偏度 | 需要 n |
|---------|------|--------|
| 均匀分布 | 0（对称）| n≥5 |
| 指数分布 | 2（右偏）| n≥30 |
| 对数正态（σ=1）| 6.2 | n≥100 |
| 对数正态（σ=2）| 414 | n≥1000+ |

**结论**：n=30 是经验法则，不是魔法数字。对于严重偏态分布，需要更大的 n。

## 方法

1. 选择三个不同形状的原始分布：均匀、指数、二项
2. 对每个分布，分别设置样本量 n = 1, 5, 30, 100
3. 对每个 n，重复抽样 10000 次，每次计算样本均值
4. 绘制 10000 个样本均值的直方图
5. 叠加理论的正态分布密度曲线
6. 观察：随着 n 增大，直方图越来越接近正态曲线

## 洞察

1. **n=30 不是魔法**。它是一个工作良好的经验法则，但对于偏态分布（如指数分布），n=30 时均值分布"有点正态但尾部还有一点偏"。对于对数正态（σ=2），n=100 还不够。

2. **CLT 解释了两个关键事实**：
   - 为什么正态分布无处不在（很多自然量都是大量微小效应之和）
   - 为什么 A/B 测试有效（样本均值近似正态→可以用 z 检验）

3. **CLT 的局限**：
   - 要求方差有限（Cauchy 分布就没有均值→CLT 不可用）
   - 要求独立同分布（时间序列数据通常不独立）
   - 样本量需要根据原始分布的偏度调整

4. **CLT 的实战含义**：
   - 你不必知道需求的确切分布来估计平均需求
   - 你只需要样本量够大，样本均值就是（近似）正态的
   - 这就是为什么你可以用正态分布来做置信区间，即使原始数据不是正态的
