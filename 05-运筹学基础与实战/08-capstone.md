# 🏆 毕业项目：智能供应链网络设计

> **难度**：★★★★☆ [综合实践]
> **组合技术**：工厂选址（MIP）+ 运输路径优化（LP/网络流）+ 需求不确定性（仿真）
> **前置要求**：至少完成案例1（物流）和案例2（库存），熟悉建模三要素
> **预计耗时**：4-8 小时

---

## 8.1 项目背景

你是一家大型制造企业的运筹学团队负责人。公司准备在全国建设新的供应链网络，CEO 给你以下信息：

- **工厂候选地**：10 个城市（每个投资成本不同，产能上限不同）
- **配送中心**：5 个现有仓库（按区域覆盖）
- **客户群**：50 个城市（每个城市的需求量已知，但有波动）
- **运输方式**：公路 + 铁路（不同成本、不同容量）
- **时间范围**：未来 3 年，每年需求预计增长 5-15%

CEO 要求你回答三个问题：

1. **在哪建厂？** 从 10 个候选地中选择一个子集
2. **货怎么运？** 从工厂→配送中心→客户，最小化总成本
3. **如果需求变了怎么办？** 你的方案对需求波动有多敏感？

## 8.3 子问题 1：工厂选址（MIP）

### 决策变量

```
build[f] ∈ {0, 1}     是否在候选地 f 建厂
produce[f] ≥ 0        工厂 f 的年产量（连续变量）
```

### 约束

1. **产能上限**：`produce[f] ≤ capacity[f] * build[f]`（Big-M 约束）
2. **投资预算**：`Σ build[f] * invest_cost[f] ≤ budget`
3. **最低产能**：`Σ produce[f] ≥ total_demand * 1.1`（10% 安全余量）
4. **最少建厂数**：`Σ build[f] ≥ min_factories`
5. **最多建厂数**：`Σ build[f] ≤ max_factories`

### 目标

```
min  Σ (invest_cost[f] * build[f] / amort_years + operate_cost[f] * produce[f])
```

其中 `amort_years` 是投资折旧年限（假设 10 年）。

### 验证标准

- 所有约束满足
- 建厂数量在 min~max 之间
- 总产能 ≥ 需求 × 1.1

## 8.4 子问题 2：运输优化（LP/网络流）

### 决策变量

```
flow[f][w][c] ≥ 0     从工厂 f 经仓库 w 到客户 c 的运量
```

或者更简洁地分为两段：
```
to_wh[f][w] ≥ 0        工厂 f → 仓库 w 的运量
to_cust[w][c] ≥ 0      仓库 w → 客户 c 的运量
```

### 约束

1. **工厂产能**：`Σ_w to_wh[f][w] ≤ produce[f]`
2. **仓库容量**：`Σ_c to_cust[w][c] ≤ wh_capacity[w]`
3. **仓库流量守恒**：`Σ_f to_wh[f][w] = Σ_c to_cust[w][c]`
4. **客户需求满足**：`Σ_w to_cust[w][c] = demand[c]`

### 目标

```
min  Σ_{f,w} to_wh[f][w] * cost_fw[f][w] + Σ_{w,c} to_cust[w][c] * cost_wc[w][c]
```

### 验证标准

- 所有约束满足
- 运输成本 ≤ 经验方案运输成本
- 每吨公里成本合理

## 8.5 子问题 3：敏感性分析（仿真）

### 假设

- 每个城市的需求量服从正态分布：`demand[c] ~ N(μ_c, σ_c)`
- 其中 σ_c = 0.15 × μ_c（15% 标准差）
- 各城市需求之间不完全独立（设相关系数 0.3）

### 仿真流程

```
For 1000 次:
    1. 采样生成 50 个城市的需求向量
    2. 固定建厂方案（子问题 1 的结果）
    3. 重新求解运输问题（子问题 2）
    4. 记录：总成本、运输成本、是否可行（产能是否够）

输出：
    - 总成本分布（均值、P10、P50、P90）
    - 方案不可行的概率
    - 哪个参数对成本最敏感（Sobol 指标或单变量扫描）
```

## 8.6 整合模型与验证

### 验证流程

```
Step 1: 运行子问题 1（MIP 工厂选址）
        → 输出建厂方案
Step 2: 运行子问题 2（LP 运输优化）
        → 输出最优运量 + 运输成本
Step 3: 运行子问题 3（仿真 1000 次）
        → 输出不确定性分析
Step 4: 对比
        → 总成本 ≤ 经验方案的 85%
        → 方案在 95% 以上的需求场景下可行
        → 敏感性雷达图
```

### 经验方案（对照组）

同行一贯的做法是：

- 在人口最密集的 3 个城市建厂（不考虑成本差异）
- 按最短距离分配运输（不考虑仓库容量和整合运输）
- 不预留安全余量

你的 MIP 方案应该至少比经验方案便宜 15%。

## 8.8 评分标准

| 维度 | 优秀 (A) | 合格 (B) | 需改进 (C) |
|------|---------|---------|-----------|
| 子问题 1 | MIP 模型完整，Big-M 约束正确，预算/产能约束严格 | 模型基本完整，有少量线性化错误 | 约束缺失或线性化明显错误 |
| 子问题 2 | LP 模型正确，流量守恒严格，目标线性 | 基本正确但缺少某类约束 | 量纲不一致或约束逻辑错误 |
| 子问题 3 | 1000 次仿真，分布统计完整，敏感性雷达图 | 仿真次数够但缺少敏感性分析 | 仿真逻辑有缺陷 |
| 整合对比 | 优化方案 vs 经验方案全面对比，节省 ≥15% | 有对比但不够完整 | 缺少对照或对比不合逻辑 |
| 代码质量 | 模块化、可复现、注释完整 | 能跑通但可读性一般 | 不能直接运行 |

## 8.9 扩展任务（可选，加分项）

1. **多商品流**：如果工厂生产多种产品（A/B/C），每种产品运输成本不同，模型怎么调整？
2. **库存集成**：在每个仓库加入安全库存成本，把库存持有成本纳入总成本
3. **碳约束**：加入碳排放上限，比较碳税 vs 碳交易的经济影响
4. **动态规划**：如果每年做一次建厂决策（而不是一次性决定），用多阶段随机规划
5. **可视化**：用 matplotlib 或 plotly 画出供应链网络地图 + 流量热力图
