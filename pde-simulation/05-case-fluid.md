# 案例 3：管道内流体流动（Burgers 方程入门）★★★☆☆

> 在工程中，真正的 Navier-Stokes 方程求解非常复杂。但我们可以从它的 1D 简化版——**Burgers 方程**——入手，理解对流、扩散和非线性这些核心概念。本案例不求解完整的 NS 方程，而是通过 Burgers 方程让你摸到流体力学的灵魂。

---

## 场景

**管道内不可压缩层流**。想象一个长直管，流体从左向右流动。在入口处，流速有一个分布（比如正弦波形式的扰动），随着流体向下游运动：

- 如果流体黏性很大（低雷诺数），扰动会逐渐被抹平——层流稳定
- 如果流体黏性很小（高雷诺数），非线性对流效应会使波前变陡，甚至形成激波

> 这是我们能用 1D Burgers 方程模拟的简化版「管道流动」。它抓住了 NS 方程最本质的两个特征：**对流**和**扩散**。

---

## 物理建模

### 完整的 Navier-Stokes 方程（不可压缩）

完整的 NS 方程为：

```
∂u/∂t + (u·∇)u = -∇p/ρ + ν∇²u + f
```

其中左边第二项是**对流项**（非线性），右边第二项是**扩散项**（黏性）。

对于一维无压力梯度的简化情况，我们得到 **Burgers 方程**：

```
∂u/∂t + u·∂u/∂x = ν·∂²u/∂x²
```

### 各项物理含义

| 项 | 名称 | 物理含义 |
|----|------|---------|
| ∂u/∂t | 非稳态项 | 流速随时间的变化率 |
| u·∂u/∂x | 对流项（非线性） | 流体携带自身动量向下游传播 |
| ν·∂²u/∂x² | 扩散项（黏性） | 黏性使速度梯度被抹平 |

### 雷诺数

雷诺数 Re 是对流力与黏性力的比值：

```
Re = U·L / ν
```

- **Re 小**（ν 大）：黏性主导，流动稳定、易被抹平 → 层流
- **Re 大**（ν 小）：对流主导，容易出现激波、涡等复杂结构 → 可能转捩为湍流

> 在本案例中，我们通过调节 ν 来观察不同雷诺数下的行为。这是一种直观理解「什么决定了流动形态」的方式。

---

## 数值方法：有限差分法 (FDM)

### 空间离散

将空间域 x ∈ [0, L] 均匀划分为 N 个网格点，间距 Δx = L/(N-1)。

### 对流项：迎风格式 (Upwind)

对于对流项 u·∂u/∂x，不能简单使用中心差分——因为对流有方向性。迎风格式的核心思想是：

- 流体从左往右流（u > 0）→ 用后向差分
- 流体从右往左流（u < 0）→ 用前向差分

数学表达：

```python
if u[i] >= 0:
    du_dx = (u[i] - u[i-1]) / dx   # 后向差分
else:
    du_dx = (u[i+1] - u[i]) / dx   # 前向差分
```

**为什么迎风？** 因为信息沿流动方向传播，从上游获取的信息才能反映物理因果性。中心差分在对流占优时会导致非物理振荡。

### 扩散项：中心差分

对于黏性项 ν·∂²u/∂x²，使用二阶中心差分（没有方向性问题）：

```python
d2u_dx2 = (u[i-1] - 2*u[i] + u[i+1]) / dx²
```

### 时间推进

显式 Euler 格式（前向差分）：

```
u^{n+1}_i = u^n_i + Δt · [-u^n_i · (∂u/∂x)_upwind + ν · (∂²u/∂x²)_center]
```

### 稳定性条件（CFL）

| 条件 | 公式 | 含义 |
|------|------|------|
| 对流 CFL | max|u|·Δt/Δx ≤ 1 | 信息在一个步长内不能跨过多个网格 |
| 扩散 CFL | ν·Δt/Δx² ≤ 0.5 | 黏性扩散的时间步长限制 |

两个条件必须同时满足。网格加密时，扩散条件的限制更严格（Δx² 关系）。

## 运行结果

### 1. 基本求解（正弦波初始条件）

初始条件：`u(x,0) = sin(πx/L)`，两端固定为 0。

当 ν=0.02 时，可以看到：
- 正弦波逐渐被对流「扭曲」——波前变陡（非线性效应）
- 同时黏性又在不断抹平尖峰
- 最终形成一种「竞争」后的平衡形状

### 2. 不同雷诺数对比

运行 `compare_reynolds()` 可得四张子图：

| ν | Re ≈ 1/ν | 观察结果 |
|---|----------|---------|
| 0.10 | 10 | 黏性主导，波形几乎被完全抹平 |
| 0.05 | 20 | 黏性仍较强，波形显著衰减 |
| 0.02 | 50 | 对流与黏性相当，波前明显变陡 |
| 0.01 | 100 | 对流主导，激波特征显现 |

### 3. 时间演化过程

运行 `time_evolution()` 展示同一参数下不同时刻的波形：

```
                    t=0.00s
  u ↑         ~~~~~~~
    |        ~       ~
    |       ~         ~

  u ↑      _____
    |     /     \
    |    /       \
    +----------------------→ x
```

波前逐渐变陡（非线性），同时峰值在衰减（黏性）。

运行 `shock_evolution()` 使用间断初始条件（阶跃函数），观察激波如何被黏性「抹平」成连续过渡——这正是物理世界中不存在真正数学间断的原因。

## 洞察

1. **非线性是「自我强化」的**：对流项 u·u_x 意味着速度大的地方变化也大——大速度波会自我挤压变陡，最终形成激波。线性方程做不到这一点。

2. **黏性是「破坏者」也是「救星」**：黏性抹平梯度，但同时也让数值计算稳定。没有黏性，间断会无限陡——数值方法会炸。

3. **Burgers 是 NS 的完美简化**：它保住了对流、扩散、非线性这三个 NS 的核心要素，却只有一维——你可以用 FDM 轻松求解并在终端上理解它的行为。

4. **迎风 vs 中心差分**：在对流占优时（高 Re），中心差分会产生非物理振荡——迎风格式牺牲了一些精度，但换来了稳定性。这个权衡在流体力学中无处不在。

5. **工程启示**：真实 CFD 中，处理高雷诺数流动的关键就是「如何稳定地处理对流项」。迎风、TVD、WENO——所有这些格式本质上都是在回答同一个问题：如何在保稳定的同时不牺牲太多精度？
