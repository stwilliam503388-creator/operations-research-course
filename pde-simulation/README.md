# 物理方程求解基础课程

> 一个周末，从零到能理解偏微分方程是什么、物理问题怎么建模、以及用数值方法求解真实世界的物理现象。

---

## 这份教程是什么

这是一份给**有高等数学基础（微积分、线性代数）、会写 Python、但对偏微分方程和物理模拟完全陌生**的工科/技术人写的自学教程。它不讲虚的——每一章都围绕「这个方程描述了什么物理现象」和「这个东西在工程中到底在干什么」展开。读完以后，你不一定会成为计算物理学家，但你一定会：

- 跟同事解释清楚「偏微分方程是干什么的、为什么物理世界需要它来描述」
- 看懂有限差分、有限元的基本思路，知道什么时候该用哪种方法
- 用 Python + 数值库把工程中的 PDE 问题离散化并求解

## 术语速查表

| 中文 | English | 一句话人话 |
|------|---------|-----------|
| 偏微分方程 | Partial Differential Equation (PDE) | 描述一个物理量在空间和时间上如何变化的数学方程。像天气预报——告诉你未来的温度分布 |
| 阶数 | Order | 方程中最高阶偏导数的阶数。一阶=只关注变化率，二阶=关注变化率的变化率 |
| 线性 PDE | Linear PDE | 未知函数和它的导数都是线性的——好解，可以用叠加原理 |
| 非线性 PDE | Nonlinear PDE | 未知函数和它的导数之间有乘积——难解，真实世界大多是这种 |
| 椭圆型方程 | Elliptic PDE | 像一张拉紧的膜，每个点的值由周围所有点共同决定（稳态热传导） |
| 抛物型方程 | Parabolic PDE | 像一个慢慢冷却的咖啡杯——时间向前走，温度逐渐均匀（瞬态热传导、扩散） |
| 双曲型方程 | Hyperbolic PDE | 像敲鼓时声波传播——信息以有限速度沿着特征线传播（波动方程、激波） |
| 边界条件 | Boundary Condition (BC) | 物理问题在边界上已知的信息——边界温度已知？边界固定不动？ |
| 初始条件 | Initial Condition (IC) | 时间相关问题的起始状态——t=0 时的温度场是什么样的？ |
| Dirichlet 边界 | Dirichlet BC | 边界上的函数值已知——比如锅边温度固定为 100°C |
| Neumann 边界 | Neumann BC | 边界上的函数梯度已知——比如锅边是绝热的（热流量为零）|
| Robin 边界 | Robin BC | 边界上函数值和梯度的组合已知——比如表面对流换热 |
| 有限差分法 | Finite Difference Method (FDM) | 用相邻点的差商代替导数——最简单的数值离散方法 |
| 有限元法 | Finite Element Method (FEM) | 把区域切成小单元，在每个单元上假设解的形式——适合复杂几何 |
| 有限体积法 | Finite Volume Method (FVM) | 把区域切成小控制体，在每个体上保证守恒——流体力学的最爱 |
| 网格 | Mesh / Grid | 把连续的空间切成离散的小块——分辨率越高越精确但也越慢 |
| 稳定性 | Stability | 数值解不会随着时间步进越跑越偏——不稳定的方法会爆炸 |
| 收敛性 | Convergence | 网格越细，数值解越接近真实解 |
| CFL 条件 | CFL Condition | 时间步长不能太大，否则信息跑得比网格快——算法会炸 |
| 刚度 | Stiffness | 系统中不同物理过程的时间尺度相差巨大——你必须用很小的步长才能跟上最快的过程 |
| 特征线 | Characteristic Line | 信息在 PDE 中传播的路径——波动沿特征线走，扩散没有明显的特征线 |

> 读到任何一个术语忘了什么意思，回到这里看「一句话人话」列，3 秒搞定。

## 预备知识

- **高等数学**：微积分（偏导数、梯度、散度、旋度）、线性代数（矩阵运算、特征值）
- **Python 基础**：能写循环和函数，会用 NumPy 数组操作
- **基础物理**：知道热量从高温区流向低温区、力能让物体变形、波以有限速度传播
- **不需要任何 PDE 或数值分析基础**——本教程从零定义每一个概念
