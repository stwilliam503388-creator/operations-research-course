# 第一部分：概率基础

> 不讲测度论。只讲「随机变量到底在描述什么、期望和方差在决策中怎么用、大数定律为什么让你相信蒙特卡洛」。

---

## 1.1 没有概率论之前，世界是什么样子的

先想象三个场景。

**场景 A：报童决策**。你每天进 100 份报纸，卖不完亏本，卖不够少赚。需求有时 80，有时 120。你怎么办？没有概率——凭感觉进货，靠运气吃饭。有概率——你知道需求服从 N(100,30)，算出最优订货量是 113 份，期望利润最大。

**场景 B：A/B 测试**。你的 APP 改了按钮颜色后转化率从 5% 升到 5.3%。这是真有效还是随机波动？没有统计——老板拍板"上！"。有统计——p=0.03 < 0.05，差异显著，放心上线。

**场景 C：投资组合**。你有 100 万，股票年收益均 12%、标准差 20%。一年后最差能亏多少？没有概率——"应该能赚吧"。有概率——95% 置信区间是 [-27.2%, 51.2%]，你有 2.5% 的概率亏超过 27%。

**概率论不是数学——它是量化不确定性的语言。**

## 1.2 概率公理与条件概率

### 三要素

```
P(Ω) = 1              ← 所有可能情况的概率之和 = 1
0 ≤ P(A) ≤ 1          ← 任何事件的概率在 0 和 1 之间
P(A∪B) = P(A) + P(B)  ← 互斥事件的概率相加
```

### 条件概率

**直觉**：已知今天下雨了，明天下雨的概率是多少？这就是条件概率。

```
P(A|B) = P(A∩B) / P(B)

读作：在 B 发生的条件下 A 发生的概率
```

**生活类比**：全中国得某种病的概率是 0.1%。但如果你检测阳性了，得病的概率是多少？这个**从先验（0.1%）到后验（检测阳性后）**的更新，就是贝叶斯定理。

### 贝叶斯定理（先记住这个，后面案例3会反复用）

```
P(A|B) = P(B|A) × P(A) / P(B)
```

**生活类比**：你怀疑同事在摸鱼（A）。你看到他杯子在桌上但人不在（B）。贝叶斯定理告诉你：**看到证据后应该更新你的信念。**

### 全概率公式

```
P(B) = P(B|A₁)P(A₁) + P(B|A₂)P(A₂) + ... + P(B|Aₙ)P(Aₙ)
```

这是我们计算"总阳性率"的基础——当你不知道一个人是否得病时，阳性概率 = 得病时阳性的概率 × 得病率 + 没病时阳性的概率 × 没病率。

## 1.3 随机变量与分布

### 随机变量：把随机性映射成数字

```
掷骰子：X = 点数  →  P(X=1)=1/6, P(X=2)=1/6, ...
买股票：R = 收益率 → 服从正态分布 N(0.12, 0.20)
```

随机变量分两种：

- **离散随机变量**：取值是有限个或可数无穷个（骰子点数、顾客数量）
- **连续随机变量**：取值是某个区间内的任意实数（身高、收益率、等待时间）

### 离散 vs 连续

| | 离散 | 连续 |
|--|------|------|
| 取值 | 有限或可数 | 区间内的任意值 |
| 概率描述 | PMF: P(X=x) | PDF: f(x)，面积=概率 |
| 累积 | CDF: F(x) = P(X≤x) | CDF: F(x) = ∫f(t)dt |
| 例 | 二项、泊松 | 正态、指数、均匀 |

**重要！PDF 的值可以大于 1**。因为 PDF 不是概率，是"密度"。概率是 PDF 曲线下的面积。如果 PDF 在某点的值是 2，只要它的分布范围很窄（比如 0 到 0.5），总面积仍然是 1。

### 你必须认识的 6 个分布

| 分布 | 用途 | 参数 | 期望 | 方差 |
|------|------|------|------|------|
| 伯努利 | 一次试验成功/失败 | p | p | p(1-p) |
| 二项 | n次独立试验成功次数 | n,p | np | np(1-p) |
| 泊松 | 单位时间内事件发生次数 | λ | λ | λ |
| 均匀 | [a,b]内等可能 | a,b | (a+b)/2 | (b-a)²/12 |
| 正态 | 自然、测量、误差 | μ,σ² | μ | σ² |
| 指数 | 等待时间、寿命 | λ | 1/λ | 1/λ² |

### 正态分布的 68-95-99.7 法则

对于任何正态分布 N(μ, σ²)：
- 68% 的数据在 μ ± σ 内
- 95% 的数据在 μ ± 2σ 内
- 99.7% 的数据在 μ ± 3σ 内

**实战意义**：如果需求服从 N(100, 30)，你可以直接说"95% 的情况下需求在 40 到 160 之间"。不需要查表。

### 指数分布的无记忆性

指数分布有一个诡异的性质：

```
P(X > s + t | X > s) = P(X > t)
```

**人话**：你已经等了 10 分钟公交车，再等 5 分钟的概率和刚来车站等 5 分钟的概率一样。公交车"不记得"你已经等了多久。

这对排队论非常重要——服务时间通常假设为指数分布。

## 1.4 期望、方差与相关性

### 期望：长期平均

```
离散：E[X] = Σ x·P(X=x)
连续：E[X] = ∫ x·f(x)dx
```

**生活类比**：掷骰子 1 次你不知道结果。掷 1 万次，平均值一定接近 3.5。3.5 就是期望。

### 期望的线性性质（超级重要）

```
E[aX + bY] = aE[X] + bE[Y]
```

无论 X 和 Y 是否独立，这都成立。这意味着你不需要知道两个变量的关系就能算组合的期望。

### 方差：风险的度量

```
Var(X) = E[(X - μ)²] = E[X²] - E[X]²
标准差 σ = √Var(X)
```

**在运筹学中**：期望度量"平均表现"，方差度量"风险"。投资组合优化就是在给定的期望收益下最小化方差。

方差的计算公式 E[X²] - E[X]² 非常实用——很多时候算 E[X²] 比算 (X-μ)² 更容易。

### 协方差与相关系数

```
Cov(X,Y) = E[(X-μx)(Y-μy)]
ρ = Cov(X,Y) / (σx·σy)
```

- ρ = 1：完全正相关（油价和汽油价）
- ρ = -1：完全负相关（雨伞销量和冰淇淋销量）
- ρ = 0：不相关（但不独立）

**注意**：ρ=0 不代表独立。比如 Y = X²，X 在 [-1,1] 上均匀分布——X 和 Y 明显相关（Y 由 X 决定），但 Cov(X, X²) = 0。

**实战意义**：在投资组合中，如果你选两只负相关的股票，组合的方差会小于各自方差之和——这就是分散投资的数学基础。

### 方差的性质

```
Var(aX + bY) = a²Var(X) + b²Var(Y) + 2ab·Cov(X,Y)
```

如果 X 和 Y 独立，Cov(X,Y)=0，则 Var(aX + bY) = a²Var(X) + b²Var(Y)。

## 1.5 大数定律与中心极限定理

### 大数定律

> 样本均值以概率收敛到真实期望。

**直觉**：掷硬币 10 次可能 7 正 3 反（70% 正）。掷 1 万次，正面比例必然接近 50%。这就是为什么你相信蒙特卡洛。

**弱大数定律**（更常用）：

```
P(|X̄ₙ - μ| > ε) → 0  (n → ∞)
```

**在大数据时代**：为什么 Google 不需要精确的抽样？因为数据量够大时，样本均值=总体均值。但这有个陷阱——如果数据有系统性偏差（比如只统计了 Chrome 用户），再多的数据也没用。

### 中心极限定理

> 不管原始分布是什么，样本均值的分布趋近正态。

**直觉**：你从一个分布（不管它有多奇怪）里抽样 30 次，算平均值。把这 30 个平均值画成直方图——它一定是钟形曲线。**这就是 CLT 的魔力。**

```
X̄ ~ N(μ, σ²/n)  近似，当 n 足够大时
```

**在实战中**：你不需要知道需求的真实分布——只要样本量够大，样本均值就是正态的。这就是几乎所有统计方法的理论基础。

**重要条件**：
1. 样本独立同分布（i.i.d.）
2. 方差有限（排除了 Cauchy 分布等厚尾分布）
3. n 足够大（经验法则 n≥30，但对严重偏态分布可能需要更大）

## 1.6 一些重要不等式

这些不等式在理论证明中频繁出现，但实战中你只需要记住它们的存在。

### 马尔可夫不等式

```
P(X ≥ a) ≤ E[X]/a    对 X ≥ 0, a > 0
```

### 切比雪夫不等式

```
P(|X - μ| ≥ kσ) ≤ 1/k²
```

**人话**：任何分布的数据，距离均值 k 个标准差的概率不超过 1/k²。
- k=2：最多 25% 的数据在 2σ 之外
- k=3：最多 11.1% 的数据在 3σ 之外

对比正态分布：正态只有 5% 在 2σ 外、0.3% 在 3σ 外——所以切比雪夫给出的上界非常宽松。
