# 概率论与数理统计基础课程

> 一个周末，从零到能理解概率论的核心概念、看懂随机变量的分布、会用蒙特卡洛仿真评估不确定性——为运筹学、博弈论、随机规划打下数学基础。

---

## 这份教程是什么

这是一份给**有高中数学基础、正在学习OR/博弈论/随机规划、但概率知识停留在"背公式"阶段**的技术人写的自学教程。它不讲学院派的测度论——每一章都围绕「这个概率概念在优化/决策中有什么用」展开。

读完以后，你一定会：
- 区分清楚 PDF、PMF、CDF，知道什么时候用哪个
- 看到"需求服从正态分布"就能算安全库存和服务水平
- 用蒙特卡洛仿真评估一个随机决策方案的风险
- 理解大数定律和中心极限定理为什么是统计学的灵魂

## 术语速查表

✅ **核心术语**

| 中文 | English | 一句话人话 |
|------|---------|-----------|
| 样本空间 | Sample Space | 所有可能结果的集合——掷骰子{1,2,3,4,5,6} |
| 概率 | Probability | 某个结果发生的可能性，0=不可能，1=一定发生 |
| 条件概率 | Conditional Probability | 已知B发生的情况下A发生的概率 |
| 随机变量 | Random Variable | 把随机结果映射成数字——掷骰子的点数 |
| 概率质量函数 | PMF | 离散随机变量取每个值的概率 |
| 概率密度函数 | PDF | 连续随机变量在某点的"密度"（不是概率，是密度）|
| 累积分布函数 | CDF | 随机变量 ≤ x 的概率 |
| 期望 | Expectation | 长期平均——掷10000次骰子的平均值≈3.5 |
| 方差 | Variance | 数据的分散程度——赌局的风险度量 |
| 正态分布 | Normal Distribution | 自然界最常见的分布——身高、测量误差、IQ |
| 大数定律 | Law of Large Numbers | 样本越多，平均值越接近真实期望 |
| 中心极限定理 | Central Limit Theorem | 不管原始分布是什么，样本均值的分布趋近正态 |
| 假设检验 | Hypothesis Testing | 判断一个结论是否统计显著——A/B测试的核心 |
| 置信区间 | Confidence Interval | 真值有95%概率落在这个区间内 |
| p值 | p-value | 在原假设为真的情况下，观察到当前结果的概率 |

🔧 **进阶术语**

| 中文 | English | 一句话人话 |
|------|---------|-----------|
| 贝叶斯定理 | Bayes' Theorem | 看到证据后更新你的信念 |
| 协方差 | Covariance | 两个变量一起变化的趋势 |
| 相关系数 | Correlation | 标准化的协方差，在[-1,1]之间 |
| 矩母函数 | MGF | 生成所有矩的"工厂"——期望、方差都能从它推出 |
| 充分统计量 | Sufficient Statistic | 包含了数据中所有关于参数信息的统计量 |
| 极大似然估计 | MLE | 找到最可能产生观测数据的参数值 |
| 贝叶斯估计 | Bayesian Estimation | 结合先验知识和观测数据来估计参数 |
| 马尔可夫链 | Markov Chain | 未来只取决于现在，与过去无关 |

# 第一部分：概率基础

> 不讲测度论。只讲「随机变量到底在描述什么、期望和方差在决策中怎么用、大数定律为什么让你相信蒙特卡洛」。

---

## 1.1 没有概率论之前，世界是什么样子的

先想象三个场景。

**场景 A：报童决策**。你每天进 100 份报纸，卖不完亏本，卖不够少赚。需求有时 80，有时 120。你怎么办？没有概率——凭感觉进货，靠运气吃饭。有概率——你知道需求服从 N(100,30)，算出最优订货量是 113 份，期望利润最大。

**场景 B：A/B 测试**。你的 APP 改了按钮颜色后转化率从 5% 升到 5.3%。这是真有效还是随机波动？没有统计——老板拍板"上！"。有统计——p=0.03 < 0.05，差异显著，放心上线。

**场景 C：投资组合**。你有 100 万，股票年收益均 12%、标准差 20%。一年后最差能亏多少？没有概率——"应该能赚吧"。有概率——95% 置信区间是 [-27.2%, 51.2%]，你有 2.5% 的概率亏超过 27%。

**概率论不是数学——它是量化不确定性的语言。**

---

## 1.2 概率公理与条件概率

### 三要素

```
P(Ω) = 1              ← 所有可能情况的概率之和 = 1
0 ≤ P(A) ≤ 1          ← 任何事件的概率在 0 和 1 之间
P(A∪B) = P(A) + P(B)  ← 互斥事件的概率相加
```

### 条件概率

**直觉**：已知今天下雨了，明天下雨的概率是多少？这就是条件概率。

```
P(A|B) = P(A∩B) / P(B)

读作：在 B 发生的条件下 A 发生的概率
```

**生活类比**：全中国得某种病的概率是 0.1%。但如果你检测阳性了，得病的概率是多少？这个**从先验（0.1%）到后验（检测阳性后）**的更新，就是贝叶斯定理。

### 贝叶斯定理（先记住这个，后面案例3会反复用）

```
P(A|B) = P(B|A) × P(A) / P(B)
```

**生活类比**：你怀疑同事在摸鱼（A）。你看到他杯子在桌上但人不在（B）。贝叶斯定理告诉你：**看到证据后应该更新你的信念。**

---

## 1.3 随机变量与分布

### 随机变量：把随机性映射成数字

```
掷骰子：X = 点数  →  P(X=1)=1/6, P(X=2)=1/6, ...
买股票：R = 收益率 → 服从正态分布 N(0.12, 0.20)
```

### 离散 vs 连续

| | 离散 | 连续 |
|--|------|------|
| 取值 | 有限或可数 | 区间内的任意值 |
| 概率描述 | PMF: P(X=x) | PDF: f(x)，面积=概率 |
| 累积 | CDF: F(x) = P(X≤x) | CDF: F(x) = ∫f(t)dt |
| 例 | 二项、泊松 | 正态、指数、均匀 |

### 你必须认识的 6 个分布

| 分布 | 用途 | 参数 | 期望 | 方差 |
|------|------|------|------|------|
| 伯努利 | 一次试验成功/失败 | p | p | p(1-p) |
| 二项 | n次独立试验成功次数 | n,p | np | np(1-p) |
| 泊松 | 单位时间内事件发生次数 | λ | λ | λ |
| 均匀 | [a,b]内等可能 | a,b | (a+b)/2 | (b-a)²/12 |
| 正态 | 自然、测量、误差 | μ,σ² | μ | σ² |
| 指数 | 等待时间、寿命 | λ | 1/λ | 1/λ² |

---

## 1.4 期望、方差与相关性

### 期望：长期平均

```
离散：E[X] = Σ x·P(X=x)
连续：E[X] = ∫ x·f(x)dx
```

**生活类比**：掷骰子 1 次你不知道结果。掷 1 万次，平均值一定接近 3.5。3.5 就是期望。

### 方差：风险的度量

```
Var(X) = E[(X - μ)²] = E[X²] - E[X]²
标准差 σ = √Var(X)
```

**在运筹学中**：期望度量"平均表现"，方差度量"风险"。投资组合优化就是在给定的期望收益下最小化方差。

### 协方差与相关系数

```
Cov(X,Y) = E[(X-μx)(Y-μy)]
ρ = Cov(X,Y) / (σx·σy)
```

- ρ = 1：完全正相关（油价和汽油价）
- ρ = -1：完全负相关（雨伞销量和冰淇淋销量）
- ρ = 0：不相关（但不独立）

---

## 1.5 大数定律与中心极限定理

### 大数定律

> 样本均值以概率收敛到真实期望。

**直觉**：掷硬币 10 次可能 7 正 3 反（70% 正）。掷 1 万次，正面比例必然接近 50%。这就是为什么你相信蒙特卡洛。

### 中心极限定理

> 不管原始分布是什么，样本均值的分布趋近正态。

**直觉**：你从一个分布（不管它有多奇怪）里抽样 30 次，算平均值。把这 30 个平均值画成直方图——它一定是钟形曲线。**这就是 CLT 的魔力。**

**在实战中**：你不需要知道需求的真实分布——只要样本量够大，样本均值就是正态的。这就是几乎所有统计方法的理论基础。

# 第二部分：统计推断

> 不讲严格的数学证明。只讲「用数据说话的四个步骤：估计→区间→检验→决策」。

---

## 2.1 点估计：猜一个数

```
样本均值：x̄ = (1/n) Σ xi      ← 估计 μ
样本方差：s² = 1/(n-1) Σ(xi-x̄)²  ← 估计 σ²
```

**极大似然估计**：找到最可能产生观测数据的参数值。

**生活类比**：你看到草地上有 10 只羊——你不会猜"总共有 1000 只"。你猜"大概 10-15 只"。这就是极大似然的直觉。

---

## 2.2 置信区间：给估计一个范围

```
均值 μ 的 95% 置信区间：
x̄ ± 1.96 × σ/√n
```

**生活类比**：民意调查说"候选人支持率 52%，误差 ±3%"。那个 ±3% 就是置信区间。

---

## 2.3 假设检验：做决策

```
H₀: 新方案无效（原假设）
H₁: 新方案有效（备择假设）

p < 0.05 → 拒绝 H₀，认为效果显著
p ≥ 0.05 → 不能拒绝 H₀，效果不显著
```

**两类错误**：
- 第一类：误报（说有效其实无效）→ 概率 α（通常 0.05）
- 第二类：漏报（说无效其实有效）→ 概率 β

---

## 2.4 贝叶斯推断

频率学派：参数是固定的，数据是随机的。
贝叶斯学派：参数是随机的，数据是固定的——我们用数据更新对参数的信念。

```
后验 ∝ 似然 × 先验
P(θ|D) ∝ P(D|θ) × P(θ)
```

**与频率学派的对比**：

| | 频率学派 | 贝叶斯学派 |
|--|---------|-----------|
| 参数 | 固定但未知 | 随机，有分布 |
| 先验 | 不需要 | 需要先验分布 |
| 结果 | p 值、置信区间 | 后验分布、可信区间 |
| 解释 | 长期频率 | 主观信念的更新 |

---

## 2.5 蒙特卡洛仿真

蒙特卡洛 = 用随机抽样来解决确定性问题。

```
估计圆周率 π：
  在正方形内随机投点 → 落在圆内的比例 ≈ π/4
```

在运筹学中：
- 库存：模拟未来需求，评估缺货概率
- 排队：模拟病人到达，评估等待时间
- 投资：模拟资产价格路径，评估组合风险
