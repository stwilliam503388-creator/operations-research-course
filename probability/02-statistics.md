# 第二部分：统计推断

> 不讲严格的数学证明。只讲「用数据说话的四个步骤：估计→区间→检验→决策」。

---

## 2.1 点估计：猜一个数

你有一组样本数据，想猜总体参数。

### 基本估计量

```
样本均值：x̄ = (1/n) Σ xi      ← 估计 μ
样本方差：s² = 1/(n-1) Σ(xi-x̄)²  ← 估计 σ²
```

为什么样本方差除以 n-1 而不是 n？因为用样本均值代替了真实均值，损失了一个"自由度"。这叫**无偏估计**——E[s²] = σ²。

### 极大似然估计（MLE）

找到最可能产生观测数据的参数值：

```
θ̂_MLE = argmax  P(数据 | θ)
```

**生活类比**：你看到草地上有 10 只羊——你不会猜"总共有 1000 只"。你猜"大概 10-15 只"。这就是极大似然的直觉。

**常见 MLE**：
- 正态分布 μ 的 MLE = 样本均值 x̄
- 正态分布 σ² 的 MLE = (1/n) Σ(xi-x̄)²（有偏，但 n 大时≈无偏）
- 伯努利 p 的 MLE = 成功次数 / 总次数

### 估计量的评判标准

| 标准 | 含义 |
|------|------|
| 无偏性 | E[θ̂] = θ（长期平均等于真值）|
| 有效性 | 方差最小的无偏估计 |
| 一致性 | n→∞时 θ̂→θ（样本越多越准）|

## 2.2 置信区间：给估计一个范围

点估计只是一个数，置信区间告诉我们这个数有多可靠。

### 均值的置信区间

```
均值 μ 的 95% 置信区间：
x̄ ± z_{α/2} × σ/√n

σ 已知时：z_{0.025} = 1.96
σ 未知时：用 t 分布（t_{0.025, n-1}）
```

**生活类比**：民意调查说"候选人支持率 52%，误差 ±3%"。那个 ±3% 就是置信区间（的一半）。

### 置信区间的解读

**正确说法**：如果重复抽样 100 次，构建 100 个 95% 置信区间，其中大约 95 个包含真实均值。

**错误说法**："真实均值有 95% 概率落在这个区间内。"（真实均值是固定的，不是随机的——区间才是随机的。）

### 什么影响置信区间的宽度

| 因素 | 变化 | 区间宽度 |
|------|------|---------|
| 样本量 n | 增大 | 变窄 |
| 置信水平 | 提高（95%→99%） | 变宽 |
| 数据方差 σ² | 增大 | 变宽 |

## 2.3 假设检验：做决策

### 基本框架

```
H₀: 新方案无效（原假设/零假设）
H₁: 新方案有效（备择假设）

p < 0.05 → 拒绝 H₀，认为效果显著
p ≥ 0.05 → 不能拒绝 H₀，效果不显著
```

### 两类错误

| | H₀ 为真 | H₁ 为真 |
|--|---------|---------|
| 不拒绝 H₀ | ✓ 正确 | 第二类错误（漏报）β |
| 拒绝 H₀ | 第一类错误（误报）α | ✓ 正确（检验效力=1-β）|

- **α（显著性水平）**：通常设为 0.05——你愿意接受 5% 的误报风险
- **β**：漏报的概率——取决于样本量、效应量、α
- **检验效力 1-β**：当 H₁ 为真时正确拒绝 H₀ 的概率

### p 值的正确理解

**p 值 = P(数据 | H₀ 为真)**，不是 P(H₀ 为真 | 数据)。

**人话**：假设新方案真的无效，那看到现在这个结果的概率是 3%。这个概率很小，所以我们倾向于认为"新方案无效"这个假设可能不对。

### 常见误区

> p=0.03 不代表"新方案有 97% 的概率有效"。

p 值是条件于 H₀ 的，不是条件于数据的。要得到 P(H₁|数据)，你需要贝叶斯方法。

### 实战：A/B 测试的完整流程

1. **设定**：H₀: p_new = p_old, H₁: p_new ≠ p_old
2. **收集数据**：两组各 n 个样本
3. **计算检验统计量**：z = (p̂_new - p̂_old) / SE
4. **计算 p 值**
5. **决策**：p < 0.05 → 拒绝 H₀
6. **报告**：给出效应量（差异的置信区间）

## 2.4 贝叶斯推断

频率学派：参数是固定的，数据是随机的。
贝叶斯学派：参数是随机的，数据是固定的——我们用数据更新对参数的信念。

```
后验 ∝ 似然 × 先验
P(θ|D) ∝ P(D|θ) × P(θ)
```

### 与频率学派的对比

| | 频率学派 | 贝叶斯学派 |
|--|---------|-----------|
| 参数 | 固定但未知 | 随机，有分布 |
| 先验 | 不需要 | 需要先验分布 |
| 结果 | p 值、置信区间 | 后验分布、可信区间 |
| 解释 | 长期频率 | 主观信念的更新 |

### 贝叶斯更新的直觉

1. **先验**：你对参数最初的信念（比如"这个药大概率无效"）
2. **似然**：数据对各个参数值的支持程度
3. **后验**：看到数据后更新过的信念

随着数据增多，先验的影响逐渐减弱，后验趋近于 MLE。

### 贝叶斯 vs 频率学派：什么时候用哪个

| 场景 | 推荐 |
|------|------|
| 你有很强的领域知识 | 贝叶斯（把知识编码为先验）|
| 样本量小 | 贝叶斯（先验可以"借力"）|
| 需要自动决策 | 频率学派（p 值有明确的阈值）|
| 需要不确定性量化 | 贝叶斯（直接给出后验分布）|
| 监管要求 | 看行业惯例 |

## 2.5 蒙特卡洛仿真

蒙特卡洛 = 用随机抽样来解决确定性问题。

```
估计圆周率 π：
  在正方形内随机投点 → 落在圆内的比例 ≈ π/4
```

### 为什么它有效

大数定律保证：随着抽样次数增加，样本均值收敛到真实期望。

### 在运筹学中

- **库存**：模拟未来需求，评估缺货概率
- **排队**：模拟病人到达，评估等待时间
- **投资**：模拟资产价格路径，评估组合风险

### 收敛速度

蒙特卡洛的误差 ∝ 1/√n：
- 100 次仿真：误差 ~10%
- 10,000 次仿真：误差 ~1%
- 1,000,000 次仿真：误差 ~0.1%

要减半误差，需要 4 倍的仿真次数。这是蒙特卡洛的主要缺点（收敛慢）。

### 方差缩减技术

1. **对偶变量**：每生成一个随机数，用它和它的"镜像"（1-u）分别计算
2. **控制变量**：用一个已知期望的变量来修正估计
3. **分层抽样**：先分层再在各层内抽样

## 2.6 实战建议：统计在产品决策中的应用

### 做一个"统计素养"合格的产品经理/工程师

1. **永远问"效果量是多少"**：不要只看 p 值，要看实际差异有多大
2. **知道置信区间比点估计更有用**："转化率提升 2%" vs "转化率提升 0.5%-3.5%"
3. **先验很重要**：如果某个结果违反常识，质疑数据质量比相信 p 值更合理
4. **样本量不够时不要强推**：n<30 的 A/B 测试结果基本不可信
5. **多重比较要校正**：同时测 20 个指标，按 p<0.05 标准，平均会有 1 个虚假显著
