# 案例4：合作博弈与公平分配

> ★★★☆☆ | 沙普利值 | 核 | 边际贡献 | 联盟理性

---

## 场景

三个部门决定合作完成一个大型项目。它们各自和合作能创造的利润不同：

| 联盟 | 成员 | 利润（万元） |
|------|------|-------------|
| {A} | 研发部 | 20 |
| {B} | 市场部 | 30 |
| {C} | 运营部 | 25 |
| {A,B} | 研发+市场 | 60 |
| {A,C} | 研发+运营 | 55 |
| {B,C} | 市场+运营 | 70 |
| {A,B,C} | 全部 | 100 |

**问题**：100 万利润怎么分才算「公平」？

> 这是合作博弈的核心问题——**合作创造了额外价值，但分配方式决定合作能否维持**。

---

## 博弈建模

### 形式化

- **参与者**：N = {A, B, C}
- **特征函数** v(S)：任意联盟 S ⊆ N 可创造的利润
- **目标**：找到一个分配向量 x = (x_A, x_B, x_C)，满足：
  - **效率**：x_A + x_B + x_C = v(N) = 100
  - **个体理性**：x_i ≥ v({i})（每个人至少拿到单独干的价值）
  - **联盟理性**：Σ_{i∈S} x_i ≥ v(S)（任何子联盟至少拿到他们自己干的价值）

---

## 沙普利值

### 直觉

> 一个人的公平份额 = 他加入联盟时带来的「边际贡献」的期望值。

具体步骤：
1. 随机排列所有参与者（3! = 6 种排列）
2. 在每种排列中，依次加入参与者，计算他带来的价值增量
3. 对所有排列取平均

### 公式

```
φ_i(v) = (1/n!) × Σ_{π ∈ Π} [v(S_π(i) ∪ {i}) - v(S_π(i))]
```

其中 S_π(i) 是在排列 π 中排在 i 之前的参与者集合。

### 计算过程

以 A（研发部）为例，6 种排列：

| 排列 | 加入顺序 | 加入前联盟 | 边际贡献 |
|------|---------|-----------|---------|
| A→B→C | A 先 | {} | 20 |
| A→C→B | A 先 | {} | 20 |
| B→A→C | A 第二 | {B}=30 | 60-30=30 |
| B→C→A | A 第三 | {B,C}=70 | 100-70=30 |
| C→A→B | A 第二 | {C}=25 | 55-25=30 |
| C→B→A | A 第三 | {B,C}=70 | 100-70=30 |

φ(A) = (20+20+30+30+30+30) / 6 = 26.67

同理可得 φ(B) = 39.17，φ(C) = 34.17

---

## 代码概览

```python
def shapley_values(players, v_func):
    n = len(players)
    contributions = defaultdict(float)
    for perm in itertools.permutations(range(n)):
        coalition_mask = 0
        for idx in perm:
            prev_val = v_func(coalition_mask)
            new_mask = coalition_mask | (1 << idx)
            marginal = v_func(new_mask) - prev_val
            contributions[players[idx]] += marginal
            coalition_mask = new_mask
    n_fact = math.factorial(n)
    return {p: contributions[p] / n_fact for p in players}
```

## 洞察

1. **B 为什么拿最多？** 因为 B（市场部）的边际贡献最大（平均 39.17），B 和 C 配合也好（B+C=70）
2. **平均分配不「公平」**——平均分配（33.3, 33.3, 33.3）违反了联盟理性：{B,C} 联盟得 66.67 但 B、C 自己合作就能拿 70
3. **沙普利值是唯一满足四个公理的分配**——效率、对称性、虚设玩家、可加性
4. **核 vs 沙普利值**：核是一个集合（所有稳定分配），沙普利值是其中一种。不是所有博弈的核都非空

## 延伸

- **加权沙普利值**：某些玩家有更大的投票权重时
- **核仁（Nucleolus）**：另一种公平分配概念，最小化最大不满
- **班扎夫指数**：投票博弈中衡量权力的工具
- **应用**：成本分摊（云计算成本、供应链费用）、利润共享、碳排放权分配
