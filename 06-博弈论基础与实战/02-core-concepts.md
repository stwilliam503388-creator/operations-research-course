
# 第二部分：核心概念

> 不讲学院派的拓扑学证明。只讲「纳什均衡到底是什么、怎么找、什么时候它不唯一」。

---

## 2.1 纳什均衡：没人想单方面改变

### 直觉

纳什均衡是博弈理论的中心概念。它说的是一组策略组合，满足：

> **给定别人的选择，没有人可以通过单方面改变自己的选择得到更高的收益。**

**生活类比**：开车靠左还是靠右？在中国大家都靠右——如果有人单方面改成靠左，他会被撞飞。所以"所有人靠右"是一个纳什均衡（在靠右行驶的国家）。"所有人靠左"在靠右行驶的国家不是一个均衡——单方面改到靠右会更好。

### 找纳什均衡的下划线法

```
                奶茶店 B
              高价    低价
       高价  (5,5)  (2,8)
奶茶店 A
       低价  (8,2)  (3,3)
```

**步骤**：
1. 固定 B 的选择，看 A 的最优反应
   - B 选高价 → A 选低价更好（8 > 5）→ 在 8 下划横线
   - B 选低价 → A 选低价更好（3 > 2）→ 在 3 下划横线
2. 固定 A 的选择，看 B 的最优反应
   - A 选高价 → B 选低价更好（8 > 5）→ 在 8 下划横线
   - A 选低价 → B 选低价更好（3 > 2）→ 在 3 下划横线
3. 看哪些格子两边都被划了线 → (低价,低价) 是唯一的纳什均衡

## 2.2 占优策略

### 直觉

> **不管别人怎么选，我选这个总是最好的。**

如果存在占优策略，博弈很简单——你不需要猜别人怎么选。

**例子**：囚徒困境中的"坦白"——不管同伙选什么，坦白总是更好的。

| | 同伙沉默 | 同伙坦白 |
|--|---------|---------|
| 你沉默 | -1, -1 | -10, 0 |
| 你坦白 | 0, -10 | -5, -5 |

你的视角：同伙沉默时坦白（0 > -1），同伙坦白时坦白（-5 > -10）。**坦白是占优策略。**

## 2.3 混合策略：当纯策略不存在均衡时

有些博弈没有纯策略纳什均衡。比如**猜拳**——如果我一直出石头，你只要一直出布就赢了。所以唯一合理的策略是**随机化**：石头/剪刀/布各 1/3。

**混合策略** = 以一定概率随机选择纯策略。

### 找混合策略纳什均衡

以鹰鸽博弈为例：

| | 鸽 | 鹰 |
|--|----|----|
| 鸽 | 3,3 | 1,4 |
| 鹰 | 4,1 | 0,0 |

这只混合策略的直觉：如果我总是出鸽，你就总是出鹰（4>3）——所以我必须随机化来让你无法预测我。

均衡条件：**你的混合概率必须让对方"无差别"——他选任何纯策略的期望收益都一样。**

```

设 我出鹰的概率 = p，出鸽的概率 = 1-p

对方选鹰的期望收益 = p·0 + (1-p)·4 = 4 - 4p
对方选鸽的期望收益 = p·1 + (1-p)·3 = 3 - 2p

令两者相等：4 - 4p = 3 - 2p → 2p = 1 → p = 0.5

所以：我以 50% 概率出鹰、50% 概率出鸽时，
对方选鹰和选鸽的收益相同——她没有动机改变。
```

## 2.4 纳什均衡的存在性与多重性

### 存在性定理

纳什（1950）证明：**任何有限博弈（有限参与人、有限策略）至少存在一个纳什均衡**（可能包含混合策略）。

### 多重均衡问题

有些博弈有多个纳什均衡。例如**协调博弈**：

| | 去A店 | 去B店 |
|--|------|------|
| 去A店 | 2,2 | 0,0 |
| 去B店 | 0,0 | 1,1 |

(去A,去A) 和 (去B,去B) 都是纳什均衡。哪个会实际发生？**这就是多重均衡问题**——博弈论本身不告诉你会选哪个，你需要其他工具（聚点、文化、习俗、历史）来精炼。

### 精炼均衡的方法

| 精炼方法 | 直觉 | 适用场景 |
|---------|------|---------|
| 帕累托占优 | 选对所有人更好的那个 | 协调博弈 |
| 风险占优 | 选风险更低的那个 | 不确定时的选择 |
| 聚点 | 选文化/语言上突出的那个 | 日常协调 |
| 子博弈完美 | 剔除不可信的威胁 | 序贯博弈 |

## 2.5 序贯博弈与逆向归纳

当参与者不是同时出手，而是按顺序行动的时候——你需要在每一步考虑对方的理性反应。

### 逆向归纳

```
从博弈树的最后一步开始：
  每步都假设决策者选对自己最有利的
  然后向上推一层
```

**生活类比**：下棋时，高手不是走一步看一步——他们先想"如果我走到这里，对手会怎么应对"，然后再想自己的应对，以此类推。

### 可信威胁 vs 不可信威胁

```
你威胁："如果你降价，我就把价格降到成本以下，大家一起亏。"
但如果对方真的降价了，降到成本以下你也会亏——
这个威胁是「不可信的」，因为执行它对你也不利。
```

子博弈完美均衡（Subgame Perfect Equilibrium）正是在剔除这类不可信威胁——只有那些在每一步都理性的策略才是均衡。
