## 场景：两家奶茶店的价格战

你家楼下有两家奶茶店——**茶颜 A** 和 **茶颜 B**。它们的奶茶口味、位置几乎一样，顾客主要看价格。

两家店都在考虑明天是否降价：

| 策略选择 | 说明 |
|---------|------|
| **高价** | 卖 15 元/杯，利润空间大 |
| **低价** | 卖 12 元/杯，薄利多销 |

双方同时决策，互相不知道对方选什么。

**行业规律**（经验数据）：
- 如果都卖 15 元 → 各赚 **5000 元/天**
- 如果 A 降到 12 元，B 保持 15 元 → A 赚 **8000 元**，B 赚 **2000 元**
- 如果 B 降到 12 元，A 保持 15 元 → B 赚 **8000 元**，A 赚 **2000 元**
- 如果都降到 12 元 → 各赚 **3000 元/天**

> 核心矛盾：**每个店都想抢对方的顾客**，但合谋维持高价对双方更好。合谋能维持吗？

---

## 博弈建模

### 参与者（Players）

| 编号 | 参与者 | 角色 |
|-----|--------|------|
| 1 | 茶颜 A | 奶茶店，追求利润最大化 |
| 2 | 茶颜 B | 奶茶店，追求利润最大化 |

### 策略（Strategies）

每家店有两个策略：

```
S_A = {高价 (15元), 低价 (12元)}
S_B = {高价 (15元), 低价 (12元)}
```

### 收益矩阵（Payoff Matrix）

收益 = 每天净利润（千元），格式：(A的收益, B的收益)

```
                   茶颜 B
              高价         低价
      ┌─────────────────────────┐
高价  │  (5, 5)    │  (2, 8)   │
茶颜A │────────────┼────────────│
低价  │  (8, 2)    │  (3, 3)   │
      └─────────────────────────┘
```

**收益矩阵（数值版）**：

| A \ B | 高价 | 低价 |
|------|------|------|
| **高价** | (5, 5) | (2, 8) |
| **低价** | (8, 2) | (3, 3) |

---

## 均衡分析（下划线法）

### 第一步：找 A 的最优反应（固定 B 的选择）

**当 B 选高价时**，A 的两个策略比较：
- A 选高价 → 收益 5
- A 选低价 → 收益 8

**8 > 5** → 给 8 画下划线。

**当 B 选低价时**，A 的两个策略比较：
- A 选高价 → 收益 2
- A 选低价 → 收益 3

**3 > 2** → 给 3 画下划线。

```
                   茶颜 B
              高价         低价
      ┌─────────────────────────┐
高价  │  (5, 5)    │  (2, 8)   │
茶颜A │────────────┼────────────│
低价  │  (8, 2)    │  (3, 3)   │
      └─────────────────────────┘
         → 下划线     → 下划线
```

### 第二步：找 B 的最优反应（固定 A 的选择）

**当 A 选高价时**，B 的两个策略比较：
- B 选高价 → 收益 5
- B 选低价 → 收益 8

**8 > 5** → 给 8 画下划线。

**当 A 选低价时**，B 的两个策略比较：
- B 选高价 → 收益 2
- B 选低价 → 收益 3

**3 > 2** → 给 3 画下划线。

```
                   茶颜 B
              高价         低价
      ┌─────────────────────────┐
高价  │  (5, 5)    │  (2, 8)   │
茶颜A │────────────┼────────────│
低价  │  (8, 2)    │  (3, 3)   │
      └────下划线───┴────下划线─┘
```

### 第三步：找「两边都被下划线」的格子

| 格子 | A 下划线？ | B 下划线？ | 均衡？ |
|------|-----------|-----------|--------|
| (高价, 高价) | ❌ | ❌ | ❌ |
| (高价, 低价) | ❌ | ✅ | ❌ |
| (低价, 高价) | ✅ | ❌ | ❌ |
| **(低价, 低价)** | **✅** | **✅** | **✅ 纳什均衡** |

### 结果

> **唯一的纳什均衡是（低价, 低价），双方各赚 3000 元/天。**

```
========== 占优策略分析 ==========
A的视角：
  B选高价时：低价(8) > 高价(5)
  B选低价时：低价(3) > 高价(2)

  A选高价时：低价(8) > 高价(5)

  → 低价是B的占优策略 ✅

结论：双方都有占优策略「低价」
强制降价是各自的最佳选择

========== 帕累托差距分析 ==========
合作解（都高价）：(5, 5)，总收益 = 10
纳什均衡（都低价）：(3, 3)，总收益 = 6
帕累托效率损失：40.0%
→ 每个人都理性选择，结果比合作差了 40%

========== 重复博弈模拟（以牙还牙） ==========
回合 1: A=合作(高价) B=合作(高价) → (5,5)
回合 2: A=合作(高价) B=合作(高价) → (5,5)
...

场景：A在第2回合单方面背叛，之后恢复合作

  回合 3: A=合作, B=背叛 → (2,8)  ← B惩罚
  回合 4: A=合作, B=合作 → (5,5)  ← B原谅
  回合 5: A=合作, B=合作 → (5,5)
  ...
→ 以牙还牙既惩罚背叛，又宽容悔改
```

## 🔍 洞察

### 囚徒困境的核心矛盾

```
┌──────────────────────────────────────────┐
│                                          │
│  个人理性  ≠  集体理性                   │
│                                          │
│  每个人选对自己最好的                    │
│  结果对所有人都不好                      │
│                                          │
│  合作解 (5,5) 优于 均衡 (3,3)           │
│  但合作不是均衡 → 无法维持               │
│                                          │
└──────────────────────────────────────────┘
```

### 为什么囚徒困境无处不在？

现实中几乎所有「价格战」、「军备竞赛」、「广告战」都是囚徒困境：
- 每个航空公司都想降价抢客 → 全行业利润率趋零
- 每个国家都想增加军备 → 军费开支失控
- 每个平台都想烧钱补贴 → 全行业亏损

### 如何逃离囚徒困境？

| 方法 | 原理 | 例子 |
|------|------|------|
| **重复博弈** | 如果知道明天还会遇到对方，背叛会被报复 | 商业伙伴长期合作 |
| **合同约束** | 法律强制合作 | 反垄断协议（但违法） |
| **声誉机制** | 背叛者社死 | 电商信用评分 |
| **政府管制** | 外部力量强制 | 价格法、最低工资 |

### 以牙还牙策略（Tit-for-Tat）

阿克塞尔罗德（1984）的计算机竞赛中，最简单的「以牙还牙」策略赢得了所有锦标赛：

| 规则 | 说明 |
|------|------|
| 第一回合 | 选择合作（高价） |
| 之后每回合 | 复制对方上一回合的选择 |

它成功的原因：
1. **友善** — 从不先背叛
2. **报复** — 立即惩罚背叛
3. **宽容** — 对方恢复合作后立刻停止惩罚
4. **清晰** — 行为模式一目了然
