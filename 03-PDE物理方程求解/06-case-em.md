
# 案例 4：电磁场分布（Laplace 方程）★★★☆☆

> 平行板电容器是电磁学中最基本的器件之一。两极板之间的静电场分布由 Laplace 方程描述。本案例展示如何用 FDM + Gauss-Seidel 迭代求解 2D 静电场，并与解析解对比。

---

## 场景

**平行板电容器内部静电场分布**。

矩形区域 [0, Lx] × [0, Ly]：

```
  y ↑
    |
  Ly +----------------------+  φ = +V0
    |     （上极板）        |
    |                      |
    |     ████████████████  |
    |                      |
    |     （下极板）        |
   0  +----------------------+ → x
   0   φ = -V0            Lx
```

- 上极板 (y=Ly): 电位 φ = +V0
- 下极板 (y=0):  电位 φ = -V0
- 左右边界 (x=0, x=Lx): 电位 φ = 0（开放边界）

问题：极板之间的电位 φ(x,y) 满足什么分布？电场线如何从正极板指向负极板？

---

## 物理建模

### Laplace 方程

在没有自由电荷的区域内，静电位满足拉普拉斯方程：

```
∇²φ = 0
```

在 2D 直角坐标系下展开：

```
∂²φ/∂x² + ∂²φ/∂y² = 0
```

### 物理含义

- 电位 φ 是**调和函数**——它在任意点的值等于周围点的平均值
- **极值原理**：电位最大值和最小值一定出现在边界上（不可能是内部「凭空冒出来」的）
- 电场 E = -∇φ，电场线垂直于等位面

### 边界条件（Dirichlet 型）

| 边界 | 位置 | 条件 | 物理含义 |
|------|------|------|---------|
| 上极板 | y = Ly | φ = +V0 | 正极板电位 |
| 下极板 | y = 0  | φ = -V0 | 负极板电位 |
| 左边界 | x = 0  | φ = 0   | 开放边界 |
| 右边界 | x = Lx | φ = 0   | 开放边界 |

### 解析解（一维近似）

在平行板中心区域（忽略边缘效应），电位沿 y 方向线性变化：

```
φ(y) = V0 · (2y/Ly - 1)
```

电场均匀：E = -∂φ/∂y = -2V0/Ly（沿 -y 方向）

---

## 数值方法：FDM 五点差分 + Gauss-Seidel 迭代

### 五点差分格式

在网格点 (i,j) 处，用中心差分离散拉普拉斯算子：

```
∂²φ/∂x² ≈ (φ_{i,j-1} - 2φ_{i,j} + φ_{i,j+1}) / Δx²
∂²φ/∂y² ≈ (φ_{i-1,j} - 2φ_{i,j} + φ_{i+1,j}) / Δy²
```

令 Δx = Δy，Laplace 方程的离散形式为：

```
φ_{i-1,j} + φ_{i+1,j} + φ_{i,j-1} + φ_{i,j+1} - 4φ_{i,j} = 0
```

整理得到**五点公式**：

```
φ_{i,j} = (φ_{i-1,j} + φ_{i+1,j} + φ_{i,j-1} + φ_{i,j+1}) / 4
```

> 这恰好说明：在 Laplace 方程的解中，任何一点的值等于它四个邻居的平均值 —— 这就是「调和函数」的离散版本。

### Gauss-Seidel 迭代

Gauss-Seidel 是求解线性方程组的一种迭代方法。与 Jacobi 迭代的区别：

| 方法 | 更新公式 | 特点 |
|------|---------|------|
| Jacobi | 全部用旧值更新 | 需要两份存储，收敛慢 |
| Gauss-Seidel | 即时使用新值 | 一份存储，收敛快~2倍 |

**Gauss-Seidel 更新**：

```
φ^{k+1}_{i,j} = (φ^{k+1}_{i-1,j} + φ^{k}_{i+1,j}
                + φ^{k+1}_{i,j-1} + φ^{k}_{i,j+1}) / 4
```

注意：左边和上边的邻居已经在本轮更新过了（使用新值），右边和下边的邻居还是上一轮的旧值。

### 收敛检测

使用**相对残差**作为收敛判据：

```python
residual = max(|φ^{k+1} - φ^{k}|) / (max(φ) - min(φ))
```

当 `residual < tol`（如 10⁻⁶）时认为收敛。

## 洞察

1. **平均值的艺术**：Gauss-Seidel 的本质是反复将每个点的值设为周围四个点的平均值。这就是调和函数的离散灵魂——任何点的电位都由它的邻居决定。

2. **迭代 vs 直接求解**：对于 Laplace 方程，理论上可以用直接法解线性方程组。但对于大网格（比如 1000×1000），直接法的计算量 ~O(N³) 不可接受，而 Gauss-Seidel 每步只需 O(N²)。**迭代法是 PDE 数值求解的核心武器**。

3. **解析解 vs 数值解**：
   - 一维解析解忽略了边缘效应——它假设极板无限大
   - 数值解考虑了有限区域和左右边界的零电位条件
   - 两者在中心区域高度吻合，靠近左右边界时出现偏差——这就是边缘效应

4. **收敛速度**：Gauss-Seidel 的收敛速度与网格数有关——网格越密，需要的迭代步数越多。实际工程中会使用 SOR（超松弛迭代）加速，再加多网格法（Multigrid）突破网格数的瓶颈。

5. **从电位到电场**：EU 得到的电场满足 ∇·E = 0（无源区）和 ∇×E = 0（静电场），从数值解中计算梯度后，可以用箭矢图直观看到电场线从正极板指向负极板。

## 延伸

- **改变边界条件**：将左右边界改为 Neumann（∂φ/∂n = 0），观察边缘效应的变化
- **SOR 加速**：引入松弛因子 ω ∈ (1,2)，将 Gauss-Seidel 改为 SOR 迭代
- **复杂几何**：在区域内放置一个圆形导体（浮空电位），观察电位畸变
- **进阶方向**：求解 Poisson 方程（加入电荷密度）、时谐电磁场（Helmholtz 方程）、3D 静电场

---

> **一句话总结**：Laplace 方程 + Gauss-Seidel 迭代是最优雅的「平均值游戏」——相邻点的平均最终收敛到物理真实的电位分布。
