# 物理方程求解基础课程

> 一个周末，从零到能理解偏微分方程是什么、物理问题怎么建模、以及用数值方法求解真实世界的物理现象。

---

## 这份教程是什么

这是一份给**有高等数学基础（微积分、线性代数）、会写 Python、但对偏微分方程和物理模拟完全陌生**的工科/技术人写的自学教程。它不讲虚的——每一章都围绕「这个方程描述了什么物理现象」和「这个东西在工程中到底在干什么」展开。读完以后，你不一定会成为计算物理学家，但你一定会：

- 跟同事解释清楚「偏微分方程是干什么的、为什么物理世界需要它来描述」
- 看懂有限差分、有限元的基本思路，知道什么时候该用哪种方法
- 用 Python + 数值库把工程中的 PDE 问题离散化并求解

## 术语速查表

| 中文 | English | 一句话人话 |
|------|---------|-----------|
| 偏微分方程 | Partial Differential Equation (PDE) | 像给温度、波浪、流体写“变化说明书”：它们在空间和时间里怎么变 |
| 阶数 | Order | 看方程最高用了几层变化率：一阶看速度，二阶还看速度怎么变 |
| 线性 PDE | Linear PDE | 像能叠加的声音：两个简单解加起来还是解，通常更好处理 |
| 非线性 PDE | Nonlinear PDE | 像真实水流会互相搅动：变量之间会相乘、耦合，解起来更麻烦 |
| 椭圆型方程 | Elliptic PDE | 像一张绷紧的橡皮膜：某一点不是自己决定，而是被周围一起拉住 |
| 抛物型方程 | Parabolic PDE | 像热咖啡慢慢变凉：时间往前走，差异一点点被抹平 |
| 双曲型方程 | Hyperbolic PDE | 像敲鼓后的声波：扰动会沿着路径向外传播，而且速度有限 |
| 边界条件 | Boundary Condition (BC) | 像给问题圈边界规则：墙上温度多少、边缘能不能动、有没有热流出去 |
| 初始条件 | Initial Condition (IC) | 像按下秒表前的第一张照片：一开始温度、位移或速度是什么样 |
| Dirichlet 边界 | Dirichlet BC | 直接规定边界值，比如锅边温度就固定是 100°C |
| Neumann 边界 | Neumann BC | 规定边界“流出去多少”或斜率，比如保温墙不让热量流出 |
| Robin 边界 | Robin BC | 边界值和流量混在一起规定，像物体表面和空气交换热量 |
| 有限差分法 | Finite Difference Method (FDM) | 像用前后两个点的差，代替连续曲线的斜率 |
| 有限元法 | Finite Element Method (FEM) | 像把复杂形状切成很多小拼图，每块上用简单函数近似 |
| 有限体积法 | Finite Volume Method (FVM) | 像把空间切成一个个小盒子，每个盒子都守住“进多少、出多少” |
| 网格 | Mesh / Grid | 像给连续空间铺方格纸：格子越密越细，但计算也越重 |
| 稳定性 | Stability | 算法不会越算越疯；不稳定时，误差会像滚雪球一样爆炸 |
| 收敛性 | Convergence | 网格越细、步长越小，算出来的答案应该越来越接近真答案 |
| CFL 条件 | CFL Condition | 像规定每一步不能跨太远：信息还没传到下一格，算法别抢跑 |
| 刚度 | Stiffness | 像同一系统里既有闪电又有乌龟，快慢差太大，普通步长很难兼顾 |
| 特征线 | Characteristic Line | 像波动里的消息路线：扰动沿着这些线往前传 |

> 读到任何一个术语忘了什么意思，回到这里看「一句话人话」列，3 秒搞定。

## 预备知识

- **高等数学**：微积分（偏导数、梯度、散度、旋度）、线性代数（矩阵运算、特征值）
- **Python 基础**：能写循环和函数，会用 NumPy 数组操作
- **基础物理**：知道热量从高温区流向低温区、力能让物体变形、波以有限速度传播
- **不需要任何 PDE 或数值分析基础**——本教程从零定义每一个概念

## 学习验收标准

学完本课程后，建议用下面 5 条自检：

- 能说明椭圆型、抛物型、双曲型 PDE 分别对应什么物理现象。
- 能解释边界条件、初始条件和网格步长如何影响数值解。
- 能说清 CFL 条件为什么是稳定性约束，而不是随便调的小参数。
- 能运行 Python 热传导案例和 C++ 显式差分案例，并解释中心温度为什么会扩散下降。
- 能指出教学案例中的规则网格、简单边界在真实工程几何中什么时候不够用。

## C++ 对照案例定位
