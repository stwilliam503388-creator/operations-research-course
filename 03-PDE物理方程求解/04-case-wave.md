## 场景：地震波在地层中传播

假设在一次地震中，震源位于地下 200m 深处。地震波以 P 波（纵波）形式向各个方向传播，遇到地表和深层岩层界面时发生反射和透射。

我们简化为一维问题，研究地震波在一根 1000m 长的「地层柱」中传播：

```
地表 (x=0)                                   深层基岩 (x=L=1000m)
   │              ↑ 波传播方向                    │
   │    ╭───╮                                   │
   │   ╱     ╲    ← 震源在 x=200m 处            │
   │  ╱       ╲   高斯波包初始位移               │
   │╱           ╲                                │
   ▼                                              ▼
反射边界?                                   反射边界?
吸收边界?                                   吸收边界?
```

你的问题是：

- **地震波传播速度有多快？** — 在典型地层中约 3000 m/s
- **波遇到地表会发生什么？** — 反射回来，造成地表破坏
- **如何消除人工边界反射？** — 用吸收边界条件模拟无限域

### 物理建模

**1D 波动方程**描述了位移 u(x,t) 在介质中的传播：

$$
\frac{\partial^2 u}{\partial t^2} = c^2 \frac{\partial^2 u}{\partial x^2}
$$

| 符号 | 含义 | 典型值 |
|------|------|--------|
| u(x,t) | 质点位移 [m] | 随时间变化 |
| c | 波速 [m/s] | 地壳 P 波 ~ 3000-6000 m/s |
| x | 空间坐标 [m] | 0 ~ 1000 m |
| t | 时间 [s] | 0 ~ 若干秒 |

**初始条件**（模拟震源）：
- 位移：高斯波包 u(x,0) = A·exp(-(x-x₀)²/2σ²)
- 速度：∂u/∂t(x,0) = 0（初始静止）

**边界条件**（两种选择）：
- **反射边界**：u(0,t) = u(L,t) = 0（固定端，模拟刚性界面）
- **吸收边界**：一阶 Mur 条件，模拟波穿出计算域

## 物理方程

### 波动方程的物理含义

波动方程说的是：**某点的加速度（∂²u/∂t²）正比于该点位移分布的弯曲程度（∂²u/∂x²）**。

这跟胡克定律是一个道理——你拉一根弹簧，弹簧产生的恢复力正比于拉伸量。在连续介质中，每个「微元」受到的净恢复力正比于位移的二阶导数。

### 与热传导方程的关键区别

| 特性 | 波动方程 | 热传导方程 |
|------|---------|-----------|
| 时间导数阶数 | 二阶 | 一阶 |
| 信息传播速度 | 有限 (c) | 无限快 |
| 能量 | 守恒 | 耗散 |
| 时间可逆性 | 可逆 | 不可逆 |
| 数值稳定性 | c·Δt/Δx ≤ 1 | α·Δt/Δx² ≤ 0.5 |

### 达朗贝尔公式（解析解）

对于无限域上的 1D 波动方程，达朗贝尔给出了通解：

$$
u(x,t) = \frac{1}{2}[f(x-ct) + f(x+ct)] + \frac{1}{2c}\int_{x-ct}^{x+ct} g(\xi) d\xi
$$

其中 f(x) = u(x,0) 是初始位移，g(x) = ∂u/∂t(x,0) 是初始速度。

直观理解：**初始波形分裂成两半，一半向左走，一半向右走，速度都是 c**。

对于我们的案例（g=0）：

$$
u(x,t) = \frac{1}{2}f(x-ct) + \frac{1}{2}f(x+ct)
$$

## 数值方法：FDM 显式中心差分

### 离散化

时间二阶导数用中心差分，空间二阶导数也用中心差分：

$$
\frac{u^{n+1}_i - 2u^n_i + u^{n-1}_i}{\Delta t^2} = c^2 \frac{u^n_{i-1} - 2u^n_i + u^n_{i+1}}{\Delta x^2}
$$

整理得到**显式三时间层格式**：

$$
u^{n+1}_i = 2u^n_i - u^{n-1}_i + \sigma^2 \cdot (u^n_{i-1} - 2u^n_i + u^n_{i+1})
$$

其中：

$$
\sigma = \frac{c \Delta t}{\Delta x} \quad \text{(Courant 数)}
$$

### 第一步特殊处理

三步格式中，n=0 时 u^{-1}_i 未知。利用初始速度 ∂u/∂t(x,0) = v₀(x)：

$$
u^1_i \approx u^0_i + v_0(x_i)\Delta t + \frac{\sigma^2}{2}(u^0_{i-1} - 2u^0_i + u^0_{i+1})
$$

---

## CFL 条件验证

### 稳定性条件

$$
\sigma = \frac{c \Delta t}{\Delta x} \leq 1
$$

### 物理直觉

在一个时间步 Δt 内，波最多传播 c·Δt 的距离。这个距离不能超过一个网格间距 Δx——否则数值方法无法「看到」波的传播，会导致信息丢失和数值发散。

```
Δt 太小: 计算慢但稳定
    → → → → → → → → → → → → → → → → → → →
    ↑ 波在每个时间步只能走一点点

Δt 太大: 计算快但可能发散!
    → ← → ← → ← → ← → ← → ← → ← → ← → ← → ←
    波「跳过」了网格点，信息丢失
```

### 与热传导的稳定性对比

| 方程 | 稳定性条件 | 网格加密效应 |
|------|-----------|-------------|
| 波动 | σ ≤ 1 | Δx 减半 → Δt 减半（2倍计算量）|
| 热传导 | r ≤ 0.5 | Δx 减半 → Δt 缩到 1/4（8倍计算量）|

**这就是为什么波动方程通常用显式格式就够了**，而热传导往往需要隐式。

## 边界处理

### 反射边界（固定端）

最简单的边界：波遇到固定端会反射回来，相位翻转（位移变号）。

```python
u[n+1, 0] = 0.0
u[n+1, -1] = 0.0
```

这模拟了地震波遇到刚性基岩界面时的全反射。

### 吸收边界（一阶 Mur）

为了让波「走出」计算域而不产生人为反射，使用吸收边界条件。

**一阶 Mur 条件**：

左边界 (x=0)，波向左传播：

$$
\frac{\partial u}{\partial t} - c \frac{\partial u}{\partial x} = 0
$$

离散形式：

$$
u^{n+1}_0 = u^n_1 + \frac{\sigma - 1}{\sigma + 1} (u^{n+1}_1 - u^n_0)
$$

当 σ = 1 时这个边界是完美匹配的（波完全无反射）。σ 偏离 1 越远，残留反射越大。

### PML（完美匹配层）简介

一阶 Mur 边界对宽频带波的吸收效果有限。更先进的方法是 **PML（Perfectly Matched Layer）**，在计算域外围添加一层有损耗的介质层，波进入该层后指数衰减，几乎不产生反射。

PML 的核心思想：
1. 在 PDE 中引入坐标拉伸 + 阻尼项
2. 在边界层内逐渐增大阻尼系数
3. 波在层内衰减到几乎为零，然后遇到反射边界也无所谓了

### 反射边界

高斯波包从 x=200m 处分裂为两个波包，分别向左右传播：
- 向左的波包在 **~67ms** 到达左边界 (x=0)，反射回来
- 向右的波包在 **~267ms** 到达右边界 (x=1000m)，反射回来

反射后，波包与后续传播的波发生干涉，形成复杂的波形。

### 吸收边界

使用一阶 Mur 边界后，波包到达边界时几乎完全穿出，仅残留极小的反射（约 5-10% 振幅）。

### CFL 破坏演示

设置 σ = 1.5 (>1)：
- 前几步看起来正常
- 约 10-20 步后出现高频震荡
- 约 30-40 步后数值解完全发散（振幅指数级增长）

## 洞察

### 1. 波动方程是「可逆」的

热传导方程不能回溯时间（\* 号），但波动方程可以——因为时间导数阶数是二阶。这意味着：

- 如果你知道现在的波场和前一时刻的波场，你可以往前或往后推
- 这在地震勘探中用于**逆时偏移**（RTM）成像技术

### 2. CFL 条件的本质是因果律

CFL 条件不是数值分析家的发明——它是**物理因果律**在数值方法中的体现。数值方法只能传播一个网格间距的信息，而物理波以速度 c 传播。如果 c·Δt > Δx，数值方法就在「伪造」因果信息——这是不可能稳定的。

### 3. 反射边界 vs 吸收边界的选择

| 场景 | 边界类型 | 原因 |
|------|---------|------|
| 有限结构（吉他弦、桥梁） | 反射 | 物理边界就在这里 |
| 无限域（地震波、电磁波） | 吸收/PML | 模拟波「消失在远方」 |
| 结构动力学 | 反射 | 结构-基础相互作用 |
| 声学模拟 | 吸收 | 自由场辐射 |

## 延伸

### 1. 高阶格式

显式中心差分格式的空间精度是 O(Δx²)。可以扩展到：
- 四点差分（精度 O(Δx⁴)）
- 谱方法（精度指数级收敛，但需要周期性边界）

### 2. 2D/3D 波动方程

$$
\frac{\partial^2 u}{\partial t^2} = c^2 \left( \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} + \frac{\partial^2 u}{\partial z^2} \right)
$$

CFL 条件变为：c·Δt · √(1/Δx² + 1/Δy² + 1/Δz²) ≤ 1

### 3. 弹性波方程

真实地震波不仅包含 P 波（纵波），还包含 S 波（横波）。弹性波方程比标量波动方程复杂得多：

- 向量位移场 u = (uₓ, u_y, u_z)
- 涉及 P 波和 S 波两种波速
- 在界面处发生波型转换（P→S, S→P）

### 4. 实际地震应用

- **地震勘探**：人工震源（气枪、炸药）产生地震波，检波器接收反射波，反演地层结构
- **地震预警**：利用 P 波（快）比 S 波（慢、破坏性强）先到的特点，提前几秒发出警报
- **地震动模拟**：在已知断层破裂模型下，计算地面运动强度，指导建筑抗震设计
