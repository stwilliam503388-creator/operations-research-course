# 案例7：结构受力变形（弹性力学）★★★★☆ [进阶]

> 这一章讲梁的弯曲变形。你会看到：一个四阶 ODE（Euler-Bernoulli 梁方程）怎么用 FDM 离散化求解，四阶导数怎么拆成差分格式，固定端和自由端的边界条件怎么处理，以及数值解和材料力学教材里的解析解差多少。

---

## 场景：悬臂梁端部受力

你是一个结构工程师。一根悬臂梁（一端固定在墙上，另一端悬空）的端部受到一个向下的集中力 P，同时整根梁还承受均布载荷 q₀。你需要知道：梁会弯成什么形状？最大挠度在哪儿？会不会超过设计限值？

这是材料力学中最经典的问题。但你很快会发现：如果梁的截面沿长度变化（锥形梁）、或者载荷分布复杂、或者材料不是线弹性的——解析解就不存在了。那时候，你就得用数值方法。

## 物理建模

### Euler-Bernoulli 梁理论

Euler-Bernoulli 梁理论假设梁的横截面在变形后仍然保持平面且垂直于中性轴（即忽略剪切变形）。挠度 w(x) 满足四阶常微分方程：

$$
\frac{d^4 w}{dx^4} = \frac{q(x)}{EI}
$$

其中：
- w(x) — 梁的横向挠度 [m]
- E — 弹性模量 [Pa]
- I — 截面惯性矩 [m⁴]
- q(x) — 分布载荷 [N/m]（向下为负）

对于悬臂梁（x=0 固定端，x=L 自由端）：

| 边界 | 条件 |
|------|------|
| 固定端 (x=0) | w = 0（位移为零），w' = 0（转角为零）|
| 自由端 (x=L) | w'' = 0（弯矩为零），w''' = 0（剪力为零）|

### 为什么不是 PDE？为什么从这里开始？

严格来说，Euler-Bernoulli 梁方程是**四阶常微分方程**（ODE），不是 PDE。我们故意把它放在这里，因为：

1. **这是从 ODE 到 PDE 的完美桥梁** — 四阶意味着需要四个边界条件，这跟 PDE 的边界条件处理思路完全一致
2. **结构力学中的真实 PDE**（如板壳理论中的 Kirchhoff-Love 板方程）形式上就是：
   \[
   \nabla^4 w = \frac{q}{D} \quad \text{(双调和方程)}
   \]
   把梁的 x 方向换成 x-y 平面，就是板的 PDE
3. **解析解可以闭式求出** — 能直接对比验证数值方法的正确性

### 解析解（材料力学标准结果）

均布载荷 q₀ 和端部集中力 P 共同作用下的悬臂梁挠度：

$$
w(x) = \frac{q_0 x^2 (6L^2 - 4Lx + x^2)}{24EI} + \frac{P x^2 (3L - x)}{6EI}
$$

在端部 (x=L)：

$$
w(L) = \frac{q_0 L^4}{8EI} + \frac{PL^3}{3EI}
$$

## 数值方法：FDM 离散化四阶导数

### 四阶中心差分公式

$$
\frac{d^4 w}{dx^4} \approx \frac{w_{i-2} - 4w_{i-1} + 6w_i - 4w_{i+1} + w_{i+2}}{\Delta x^4}
$$

精度 O(Δx²)。这是五点模板（five-point stencil），每个内部节点依赖左右各两个邻居。

### 边界条件处理（关键部分）

边界条件是 FDM 处理四阶方程中最容易被坑的地方。

**固定端 (x=0)**：需要两个方程来处理 w₀ 和 w₁。

- w₀ = 0：直接设第一行系数为 1，右端项为 0
- w'(0) = 0：用二阶精度前向差分 ⇒ -3w₀ + 4w₁ - w₂ = 0

**自由端 (x=L)**：同样需要两个方程。

- w''(L) = 0 ⇒ w_{N-3} - 2w_{N-2} + w_{N-1} = 0（二阶后向差分）
- w'''(L) = 0 ⇒ -w_{N-4} + 3w_{N-3} - 3w_{N-2} + w_{N-1} = 0（二阶后向差分）

处理完后得到一个 (N×N) 的稀疏线性系统 A·w = f·Δx⁴。

### 为什么不直接用有限元？

对于一根等截面直梁，FDM 完全够用——实现简单、精度高、速度快。但在以下场景你需要 FEM：
- 变截面梁（惯性矩 I 随 x 变化）
- 复杂几何（L 型、T 型截面）
- 材料非线性或几何大变形

FDM 的局限在于：非均匀网格处理麻烦、复杂边界条件实现困难、扩展至 2D/3D 板壳理论时需要结构化网格。

## 洞察

1. **四阶问题对边界条件极其敏感** — 自由端的 w''' = 0 条件如果写错（比如漏了或精度不对），端部挠度误差可以到 10% 以上。始终用解析解验证。

2. **FDM 处理高阶导数的挑战** — 五阶模板需要每个内部节点左右各两个邻居，导致靠近边界的 2-3 个节点需要特殊处理。网格越细，边界影响的相对区域越小。

3. **收敛阶验证是调试利器** — 如果你的收敛阶明显低于理论值（这里应该是 2.0），说明某处有 bug。反之，如果收敛阶对了但绝对误差大，那是精度问题不是正确性问题。

4. **从梁到板的跳跃** — 把四阶 ODE 换成双调和方程 ∇⁴w = q/D，把五点模板换成 13 点模板（2D 情况），把边界从「两个端点」换成「四条边」——恭喜你，已经从 Euler-Bernoulli 梁学到了 Kirchhoff-Love 板。

## 延伸

- **Timoshenko 梁**：考虑剪切变形，方程为两个耦合的二阶 ODE（更适合短梁/厚梁）
- **屈曲分析**：特征值问题：d⁴w/dx⁴ + λ·d²w/dx² = 0
- **振动分析**：加上惯性项：EI·∂⁴w/∂x⁴ + ρA·∂²w/∂t² = 0
- **板弯曲**：双调和方程 ∇⁴w = q/D + 四个边界条件
- **非线性**：von Kármán 板的大变形理论
